中通快递高薪诚招各路安全人才20到45K

2017 /9/21 14:40
494
沙发

中通快递是一家集快递、物流及其他业务于一体的大型集团公司，作为中国最大的快递公司之一，一直致力于营造安全、快捷的快递服务。

2016年10月27日，中通快递成功登陆美国纽约证券交易所，创当年美国证券市场最大IPO，也是继阿里巴巴2014年赴美上市以来最大规模的中国企业赴美IPO。

中通快递信息安全部以保障上亿客户以及合作伙伴的利益为前提，通过长期的信息安全管理建设，已经搭建起一整套的信息安全管理基础、制度与人才体系。

一、安全渗透测试工程师 （15-30K）

岗位职责：

负责中通生产网和办公网的渗透测试

工作要求：

1.精通各种渗透方法、各种黑客工具，有自己挖掘漏洞的成功案例和POC编写的能力；
2.熟悉社会工程学，具备灵活的渗透思路和技巧；
3.熟悉帽子圈和黑产手法，但能保持自己的职业操守，过往无不良记录；
4.具备代码审计能力、有通用漏洞挖掘经验和大型网站实战渗透经验优先;

二、项目管理 （15-30K）

岗位职责： 

1. 负责公司信息安全类项目的实施落地；
2. 负责定期审核各项安全管理制度的执行情况并持续优化改进；
3. 负责与公司内外部的协调沟通，协助信息安全团队开展各项工作

工作要求：  

1. 具有2年以上项目管理的工作经验，涉及信息安全类项目优先；
2. 具有较强的沟通能力，善于处理好内外关系，能够承担工作压力；
3. 良好的项目规划水平及较好的文档化能力；
4. 对信息安全行业有浓厚兴趣，了解基础的信息安全知识的优先

三、 安全运营（10-15K） 

岗位职责：

1. 负责公司安全产品和服务的日常运营和推广；
2. 负责维护安全圈小伙伴的关系以及安全活动的开展；
3. 负责处理其他安全相关的日常事务

岗位要求 ：

1. 熟悉各种推广手段，如微博、微信等；
2. 良好的沟通表达能力、文字功底、团队合作能力及情绪管理能力；
3. 对信息安全行业有浓厚的兴趣；
4. 良好的外在形象和内在素质；
5. 了解基本信息安全知识的优先

四、安全开发工程师-安全风控方向（20-45K）

岗位职责：

1.负责实时风控系统的规划与开发，包括安全事件、业务事件接入，规则平台、高性能累计计算、基于spark的实时数据流分析，实时监控等， 并不断优化风控系统的扩展性、可用性和实时性；
2.负责针对不同的场景设计不同的数据采集方式，对输入风控系统的数据进行有效的监控，及时发现数据质量问题；
3.基于实时日志，利用用户行为、设备信息、账号资料等数据抽象沉淀为有效的特征及画像，并服务化供实时风控策略使用；
4.负责威胁模型的建立和风控算法、复杂策略的开发及参数调优

工作要求：

1.本科及以上学历；
2.匹配岗位的开发经验（擅长java、scala 、golang、python其中的一种或多种开发语言）；
3.对数据有较高的敏感度，了解常见的数据分析方法及算法模型；
4.学习能力强，能够无障碍阅读英文技术文档；
5.有大型互联网工作经验，实际的安全风控开发经验；
6.抗压能力强，开源社区代码贡献者优先

五、 安全开发工程师-大数据方向（20-35K）

岗位职责：

1.负责基于大数据的安全分析类产品的开发工作；
2.负责基于海量数据，建立相应的威胁模型和实时后台系统

工作要求：

1.本科及以上学历；
2.熟悉常用设计模式，具有大型分布式、高并发、高负载、高可用性系统设计开发经验；
3.有大数据平台Hadoop/Storm/Spark开发经验，擅长java 、scala、golang、 python 其中的一种或多种开发语言；
4.具有良好的抽象设计能力，思路清晰，善于思考，能独立分析和解决问题,责任心强，具备良好的团队合作精神和承受压力的能力；
5.学习能力强，能够无障碍阅读英文技术文档；
6.有大型互联网工作经验，实际的大数据处理经验；
7.接触过安全类大数据产品或者非常感兴趣，对常用的机器学习算法在安全场景下的应用有一定的了解

六、安全开发工程师-SDL方向 （15-30K）

岗位职责：

负责中通SDL（安全开发过程），以及SDL支撑工具的自动化和平台化能力建设，设计和开发相关的支撑系统和自动化工具

工作要求：

1.本科及以上学历；
2.熟悉至少一种编程语言, 如Python,shell, Java ，Golang等；
3.熟悉代码扫描工具如coverity，fortify 的使用和规则优化；
4.熟悉Linux/Windows系统原理, 并能以Linux/Mac作为工作平台；
5.熟悉devops，并可以基于devops 设计添加安全控制点；
6.有安全工具开发经验，编写过漏洞利用POC经验；
7.熟悉持续集成，自动化编译和构建平台；
8.学习能力强，能够无障碍阅读英文技术文档；

岗位信息：

部门: 信息安全部
工作类型: 全职
工作地点: 上海
