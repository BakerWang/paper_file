[招聘]上海初创团队-招10多个安全软件开发/验证工程师/经理

angelinalou 人员招聘 围观107次 [招聘]上海初创团队-招10多个安全软件开发/验证工程师/经理已关闭评论 编辑日期：2017-04-02 字体：大 中 小

2014年成立，CEO是前Intel研究院院长，公司的其他核心负责人也大都是知名外企高管背景，目前在上海新组建芯片安全团队，技术负责人是前Intel亚太研究院安全技术专家，公司获得中科院和海光半导体基金支持。

目前组建一个大概40多人的芯片安全团队-致力于x86芯片安全软件的开发， 包括x86 Linux, windows服务器和虚拟化平台。

招聘团队人员从Linux内核工程师、安全软件工程师、安全验证经理以及安全测试工程师都有需求，若有兴趣了解，可以随时邮件1845622147@qq.com或者微信：angelinalou联系. 

工作地点：漕河泾

薪水范围：工程师 15K-30K/M, 经理 30K-50K/M

Linux
Kernel developer/Linux内核开发

将专注于内核安全,虚拟化,性能,与x86 CPU的co-design

具体要求

1.
Experience on debug and patch the Linux kernel

2. 精通C语言

3. 熟悉企业Linux版本 (Oracle, Ubuntu, CentOS, SuSE)

4. 精通脚本语言, 比如. bash, python, perl.

5. 至少5年大型软件项目工作经验

6. 具有x86上的性能调试经验

7. 具有安全和加密知识的优先

8. 具有虚拟化技术的优先

9. 懂X86汇编编程的优先

Secure Software validation manager/安全软件验证经理

岗位描述： 

l  负责芯片安全软件验证团队的技术指导和管理 

任职资格： 

l  5年以上芯片安全软件领域开发验证经验 

l  熟悉jira, bugzilla, jenkins, gerrit, git, 脚本语言,有过自动化软件测试编程管理经验者优先

l  精通计算机安全领域的研发工作，熟悉各种加解密算法,比如AES, SHA, 国密; 

l  熟悉安全软件验证的整个软件流程,工具,比如code review, 功能测试,安全测试,压力测试,fuzz, ip scan, 静态代码扫描工具及分析,安全工具扫描等等

l  英语文献阅读能力

 

Secure Software engineer

岗位描述： 

l  负责安全软件的开发

任职资格： 

l  计算机相关专业背景，本科以上学历； 2年以上C语言编程经验者优先

l  熟悉linux编程. 熟悉gcc,gdb,有底层软件开发经验的和汇编开发经验的优先考虑

l  有安全(sgx, trustzone, SME, SEV)或者加解密(AES, SHA, SSL, 国密)等相关经验者优先


Secure Software validation engineer 

岗位描述：

l  负责安全软件的测试工具开发及安全软件测试 

任职资格： 

l  计算机或相关专业本科学历； 2年以上C或者脚本语言编程经验者优先

l  2年以上测试工作经验者优先； 

l  有自动化软件测试平台搭建的经验的优先

l  熟悉bugzilla, log 分析,了解regression test, escape analysis者优先

l  有安全软件测试经验者(sgx, trustzone, SME, SEV)或者加解密测试经验优先
