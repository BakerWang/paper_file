RSA Conference 2015播报之关键词

coolweis 安全资讯 围观972次 RSA Conference 2015播报之关键词已关闭评论 编辑日期：2015-05-05 字体：大 中 小

当时发布在ADLab公众号上，转到这边方便看：）

 

全球知名信息安全峰会RSA Conference 2015于2015年4月20日至4月24日在旧金山Moscone Center举办，会议主题为“Change： Challenge today's security thinking（变化：挑战当今的安全理念）”。

4月20日是RSA布展日，ADLab小编一大早就赶到了会场，由于是布展日，展厅只对参展商开放，所以人还不是很多，很多参展商大多都在忙碌着，小编不便打扰，便趁人少拍了些照片。

本次展会参展商众多，小编来之前做了些工作，提前就下载好了参展指南（下载地址：http://viewer.epaperflip.com/Viewer.aspx?docid=f787adb6-25cc-4a88-ab12-a4760163c0dc#?page=34 ）。参展指南多达220页，小编在飞机上奋战了7、8个小时终于把参展商大致翻看了一遍，并对重点关注的厂商一一做了记录待开展时再详细参观。由于参展商众多，指南中对于参展商的介绍都非常有限，小编全部看完之后发现如下出现频率较高的关键词：

APT

Zero day

Threat

Cyber Threat

Threat intelligence

Security intelligence

Security analysis

Mobile security

Application security

Cloud security

Virtualization

Visualization

Fraud

BYOD

Thread management

Behavior analytics

Big data

Machine learning

Next generation

现场看来，也确实如此，很多公司都在说intelligence，threat，visualization，cloud等等，小编将持续关注这些热门技术，看看这些公司有哪些特色的技术、产品和解决方案。

以下是一些早上趁人少时拍的现场照片，更多详情敬请期待详细报导。
