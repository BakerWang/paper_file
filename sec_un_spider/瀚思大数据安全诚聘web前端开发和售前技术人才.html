瀚思大数据安全诚聘web前端开发和售前技术人才

newland 人员招聘, 安全人才 围观277次 瀚思大数据安全诚聘web前端开发和售前技术人才已关闭评论 编辑日期：2016-10-13 字体：大 中 小

anSight瀚思致力于以大数据的收集、处理与分析技术为驱动，帮助企业实时、自动侦测已经发生或即将发生的内部与外部安全威胁，提高安全事件处理的效率，最大限度的保护企业信息资产安全。我们的创始团队均来自趋势科技、微软、Oracle等跨国公司，并拥有15年以上的IT安全领域经验。同时，作为云计算与大数据领域的连续创业者，我们在成立之初就已获得机构的投资。
现诚聘以下人才：

1、前端开发工程师（高级、中级）-北京、南京、成都、深圳
职责:

制定前端开发及协作流程，创建技术标准及规范
设计核心产品的前端架构，并带领团队成员持续的优化前端体验
与设计师、开发人员配合开发前端公共组件库，并完成详细的使用和培训文档
完成上级分派的其他任务
需求：
计算机相关专业，5年以上前端研发经验，有带团队经验者优先
深入理解W3C标准与ES规范，熟悉Web语义化，熟悉各种前端调试工具
熟悉常见前端类库，熟悉前端MVC/MVVM框架，熟悉HTML5/CSS3 等新技术，有使用AngularJS、Vue.js、React.js者优先
熟悉j2ee开发,有大型网站前端开发经验或有Echarts、D3.js等可视化开发经验者优先
技术视野广阔，有主导前端技术方案设计的能力和经验

2、售前工程师-北京、深圳
职责:

定义、撰写各类解决方案，能凸显瀚思产品价值
为客户与合作伙伴提供售前基础支持，包括系统安装、部署、故障解决与技术培训等；可以明确和并细化客户的技术需求
与客户一同设计、实施并部署瀚思产品与解决方案
深入理解客户的业务与行业特性，并能设计对应的客户网络与安全平台级方案
需求：

2年以上的技术售前或咨询经验，其中至少3年信息安全领域经验
出色的安全技术与方案功底，包括： internet/web security, firewall/IDS/IPS technologies, proxy 等
深入了解主流安全产品，如Cisco, Juniper, Palo Alto, 绿盟, 深信服, 天融信等
具备SIEM 与 SOC 经验者优先
极强的领导力、协调力与专业执行力
对解决客户的技术问题有极大的热情
优秀的写作能力，尤其是方案与标书撰写 ( word 与 ppt ) 能力，并可以将瀚思产品技术特点、商业价值与客户技术、业务需求相匹配

3、售前高级顾问-北京
职责:

定义、撰写各类解决方案，能凸显瀚思产品价值
为客户与合作伙伴提供售前基础支持，包括系统安装、部署、故障解决与技术培训等；可以明确和并细化客户的技术需求
与客户一同设计、实施并部署瀚思产品与解决方案
深入理解客户的业务与行业特性，并能设计对应的客户网络与安全平台级方案
需求：
5年以上的技术售前或咨询经验，其中至少3年信息安全领域经验
出色的安全技术与方案功底，包括：internet/web security, firewall/IDS/IPS technologies, proxy等
深入了解主流安全产品，如Cisco, Juniper, Palo Alto, 绿盟, 深信服, 天融信等
具备SIEM 与 SOC 经验者优先
极强的领导力、协调力与专业执行力
对解决客户的技术问题有极大的热情
优秀的写作能力，尤其是方案与标书撰写 ( word 与 ppt ) 能力，并可以将瀚思产品技术特点、商业价值与客户技术、业务需求相匹配
