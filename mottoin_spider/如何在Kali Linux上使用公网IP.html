如何在Kali Linux上使用公网IP

Jarry • 2017年5月2日 • 终端安全 • 阅读 848

今天通过这篇文章，您将了解攻击者如何通过匿名VPN服务来使用公共IP，这将可以在外网进行攻击。

开始吧！

在你的kali Linux中打开终端并输入以下命令启动pptp服务进行VPN配置。

apt-get install network-manager-pptp

apt-get install network-manager-pptp-gnome

Ipjetable.net为其注册用户提供免费VPN服务的网站，但本网站将通过代理服务器打开，因此我已经先用free-proxy.xyz打开ipjetable.net网页。

在这里，您需要解锁网站，以便匿名使用open vpn服务。点击关于我订阅选项卡。这里我有Google翻译

然后它需要注册解锁VPN服务现在用你的电子邮件ID进行注册。我使用临时电子邮件ID进行注册。

当您将注册到网站时，它将在您的收件箱中发送一封邮件，其中包含我们将用于VPN登录的用户名和密码。在给定的屏幕截图中，您可以观察我们突出显示链接，复制此链接。

现在上面复制了free-proxy.xyz中的链接，如上所述，为VPN服务解锁登录页面。当您解锁时，下面给出的网页会在浏览器中打开，这将要求登录凭证，就是现在通过邮件收到的用户名和密码。

现在点击安装ipjetable，启动您的本地网络的VPN服务安装。

在你的kali Linux里面点击屏幕右侧的电源图标配置VPN，然后选择有线连接。

现在点击（+）添加新的网络连接。

选择point to point tunneling protocol

现在添加用户名和密码来连接VPN服务器。然后点击“advance tab”选项卡，然后为此用户选择单选按钮存储密码。

从给出的截图选择PPTP认证和加密，然后一旦所有的配置完成复选框点击就OK。到这里，我们已经在我们的kali linux中成功配置了VPN服务。

请检查我们已经使用的vpn IP

ifconfig

从屏幕截图你可以看到192.168.0.102是我的本地ip和141.255.151.15是我们的公网IP，现在使用这个IP可以进行外网攻击。
