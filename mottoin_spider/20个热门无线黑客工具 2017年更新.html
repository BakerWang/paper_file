20个热门无线黑客工具[2017年更新]

Jarry • 2017年4月13日 • 工具 • 阅读 934

去年，我写了一篇文章，介绍流行的无线黑客工具来破解或恢复无线网络的密码。在这篇文章中，我们添加了13个工具，这些工具很受欢迎。现在我正在更新该帖子，在该列表中添加更多工具。

我没办法解释无线安全和WPA / WEP。您可以阅读有关无线黑客工具的现有文章，了解它们。在这篇文章中，我正在更新现有列表以添加更多强大的工具。我在现有列表中增加了七个新工具，为您提供了最常用的无线破解工具列表。

Aircrack

Aircrack是最流行和广为人知的无线密码破解工具。它可用于全球范围内的802.11 WEP和WPA-PSK密钥破解工具。它首先捕获网络的数据包，然后通过分析数据包尝试恢复网络的密码。它还通过一些优化来实现标准的FMS攻击，以恢复或破解网络的密码。优化包括KoreK攻击和PTW攻击，使攻击比其他WEP密码破解工具快得多。这个工具非常强大，在世界各地广泛使用。这是我将其添加到列表顶部的原因。

它提供控制台界面。如果您发现此工具难以使用，您可以尝试可用的在线教程。该工具背后的公司还提供在线教程，让您自己学习。

下载：http//www.aircrack-ng.org/

AirSnort

AirSnort是另一种流行的无线局域网密码破解工具。它可以破解Wi-Fi802.11b网络的WEP密钥。该工具基本上通过被动监视传输，然后在收集到足够的数据包时计算加密密钥。该工具免费提供给Linux和Windows平台。使用也很简单。该工具大约三年没有更新，但似乎这个工具背后的公司现在对进一步的开发感兴趣。该工具也直接涉及到WEP破解，因此被广泛应用。

下载AirSnort：http : //sourceforge.net/projects/airsnort/

Kismet

Kismet是另一款Wi-Fi 802.11 a / b / g / n第2层无线网络嗅探器和入侵检测系统。此工具基本上用于Wi-Fi故障排查。它适用于支持rfmon模式的任何Wi-Fi卡。它适用于Windows，Linux，OS X和BSD平台。该工具可以被动地收集数据包以识别标准网络，并检测隐藏网络。该工具基于客户端服务器模块化架构，可以嗅探802.11b，802.11a，802.11g和802.11n流量。它是一个开源工具，并支持最近更快的无线标准。

下载Kismet：http://www.kismetwireless.net/download.shtml

Cain & Able

Cain＆Able是另一种用于破解无线网络密码的流行工具。该工具被开发用于拦截网络流量，然后暴力破解密码。这就是为什么这个工具可以通过分析路由协议找到无线网络的密码。此工具也可用于破解其他类型的密码。它是最流行的密码破解工具之一。

这个工具不仅仅是为了WEP破解，而且还有其他各种功能。它也能用于Windows密码破解。这是这个工具在用户中如此受欢迎的原因。

下载Cain＆Able：http : //www.oxid.it/cain.html

WireShark

WireShark是网络中非常受欢迎的工具。它是网络协议分析工具，可以让您检查办公室或家庭网络中的不同内容。您可以通过检查微观级别的数据来捕获数据包并分析数据包以查找与网络相关的各种事项。此工具可用于Windows，Linux，OS X，Solaris，FreeBSD和其他平台。

如果您正在考虑尝试此工具，我建议您先阅读有关网络和协议。WireShark需要良好的网络协议知识来分析使用该工具获得的数据。如果您对此没有很好的了解，您可能找不到使用这个工具的乐趣。所以，只有当你确定你具有一定的协议知识时才能更好的使用。

Wireshark是网络中最受欢迎的工具之一，这就是为什么它被列入更高的位置。

下载Wireshark：https：//www.wireshark.org/

Fern WiFi Wireless Cracker

Fern WiFi Wireless Cracker是另一个很好的工具，可以帮助检测网络安全。它可以让您看到实时网络流量并识别主机。基本上这个工具是为了发现计算机网络的缺陷并修复出现的缺陷。它适用于苹果，Windows和Linux平台。

它能够轻松地破解和恢复WEP / WPA / WPS密钥。它还可以在基于无线或以太网的网络上运行基于网络层的攻击。对于WPA / WPA2的破解，它使用基于字典的WPS攻击。对于WEP破解，它使用Fragmentation，Chop-Chop，Caffe-Latte，Hirte，ARP Request Replay或WPS攻击。

该工具正在积极开发中。因此，您可以及时获取更新额新功能。Pro版本的工具也是可用的，提供了很多功能。

下载Fern WiFi Wireless cracker：http : //www.fern-pro.com/downloads.php

CoWPAtty

CoWPAtty是另一个不错的无线密码破解工具。它是WPA-PSK的自动化字典攻击工具来破解密码。它在Linux操作系统上运行。自带一份字典，如果密码在字典中，则此工具肯定会破解密码。但这个工具很慢，速度取决于单词列表和密码的强度。缓慢过程的另一个原因是散列使用具有SSID种子的SHA1。这意味着相同的密码将具有不同的SSIM。所以，你不能简单地使用彩虹表对所有接入点进行爆破。因此，该工具使用密码字典，并通过使用SSID为字典中包含的每个字符生成散列。该工具使用的命令很简单。

随着较新版本的工具CoWPAtty尝试通过使用预先计算的哈希文件来提高速度，以避免破解时的计算。该预先计算的文件包含大约172000个字典文件，约1000个最受欢迎的SSID。但为了成功攻击，您的SSID必须在该列表中。如果你的SSID不在1000个之中，那你完了。不过，您可以尝试这个工具来看看它是如何工作的。

下载CoWPAtty：http : //sourceforge.net/projects/cowpatty/

Airjack

Airjack是一种Wi-Fi 802.11数据包注入工具。它用于执行DOS攻击和MIM攻击。这种无线破解工具在注入伪造数据包并通过拒绝服务攻击使网络下降非常有用。该工具也可以用于网络中的中间攻击者。这个工具很受欢迎且强大的。

下载AirJack：http : //sourceforge.net/projects/airjack/

WepAttack

WepAttack是另一个工作的开源Linux工具，用于破解802.11 WEP密钥。与列表中的其他几个工具一样，此工具也会执行主动字典攻击。它从字典中测试数百万个单词以找到网络的工作密钥。只需要一个工作的WLAN卡才能使用WepAttack来执行攻击。比较有局限性，但在支持的WLAN卡上非常棒。

下载WepAttack：http : //wepattack.sourceforge.net/

NetStumbler

NetStumbler是另一种仅适用于Windows平台的无线密码破解工具。它有助于找到开放的无线接入点。这个工具是免费的。基本上，NetStumbler用于检查，验证网络配置，查找网络不良的位置，检测未经授权的访问点等。

这个工具现在不是很有效。主要原因是该工具的最后一次稳定发布是在11年前的2004年4月份。所以，它不适用于64位Windows操作系统。它也可以很容易地被检测到大多数无线入侵检测系统可用。因此，您可以使用此工具在家庭网络上学习目的，以了解其工作原理。

被称为“MiniStumbler”的修正版本也是可用的。此工具太旧，但在受支持的系统上仍然可以正常工作。所以我把它列入了这份清单。

下载NetStumbler：http : //www.stumbler.net/

inSSIDer

inSSIDer是Microsoft Windows和OS X平台最受欢迎的Wi-Fi扫描仪之一。该工具是以开源许可证发布的，也被授予“网络最佳开源软件”。后来成为高级工具，现在成本为19.99美元。inSSIDer Wi-Fi扫描器可以执行各种任务，包括查找开放的Wi-Fi接入点，跟踪信号强度，并使用GPS记录保存日志。基本上这个工具是被网络管理员用来查找无线网络中的问题

下载inSSIDer：http : //www.inssider.com/

Wifiphisher

Wifiphisher是获取无线网络密码的另一个不错的黑客工具。该工具可以对Wi-Fi无线网络执行快速自动网络钓鱼攻击来窃取密码。此工具预先安装在Kali Linux上。它可以免费使用，可用于Windows，MAC和Linux。

下载并阅读有关WiFiphisher的更多信息：https：//github.com/sophron/wifiphisher

KisMac

KisMac是与Kismet非常相似的工具，我们在上面的列表中添加。它提供与Kismet类似的功能，并被用作无线网络发现黑客工具。顾名思义，此工具仅适用于Mac。它只在受支持的无线网卡上被动地扫描网络，然后尝试通过使用暴力或利用其他漏洞来破解WEP和WPA密钥。

下载KisMac：http : //kismac-ng.org/

Reaver

Reaver是一种开源工具，用于对WPS执行强力攻击以恢复WPA / WPA2密钥。此工具托管在Google代码上，如果开发人员尚未将其迁移到另一个平台，可能会很快消失。它最近更新了大约4年前。与其他工具类似，该工具可以与使用相同攻击方法的列表中的其他工具做一个很好的替代。

下载Reaver：https : //code.google.com/p/reaver-wps/downloads/list

Wifite

Wifite也是一个很好的工具，支持通过修复程序破解WPS加密网络。它适用于基于Linux的操作系统。它也提供了与密码破解相关的各种不错的功能。

下载Wifite：https：//github.com/derv82/wifite

我们有一个关于Wifite的完整文章。阅读wifite演练。

WepDecrypt

WepDecrypt是用C语言编写的另一种无线局域网工具。该工具可以通过执行字典攻击，分布式网络攻击，密钥生成器等一些方法来猜测WEP密钥。这个工具需要很少的库来工作。您可以在下载页面上阅读更多详细信息。工具不是很受欢迎，但初学者可以看到字典攻击是如何工作的。

下载并阅读有关WepDecrypt的更多信息：http : //wepdecrypt.sourceforge.net/wepdecrypt-manual.html

OmniPeek

OmniPeek是一个数据包嗅探器和网络数据包分析工具。此工具仅适用于Windows平台，仅适用于商业用途。它还要求您对网络协议和网络数据包有很好的了解。它适用于市面上可用的大多数网络接口卡。使用可用的插件，此工具可以变得更加强大。已有大约40个插件可用于扩展此工具的功能。

下载OmniPeek：http : //www.wildpackets.com/products/distributed_network_analysis/omnipeek_network_analyzer

CloudCracker

CloudCracker是一种在线密码破解工具来破解无线网络的WPA密钥。这个工具也可以用来破解各种其他种类的密码哈希。您只需要上传握手文件并输入网络名称即可开始攻击。有3000万字长的字典，这个工具最有可能破解密码。该工具也用于MD5，SHA等几个密文的破解。如果我们谈论无线破解工具，这也是一个有效的工具，值得一提。

请参阅CloudCracker：https : //www.cloudcracker.com/

CommonView for Wi-Fi

用于Wi-Fi的CommonView也是受欢迎的无线网络监视器和打包程序分析工具。它易于理解和使用GUI来处理。这个工具基本上是针对想要监控网络相关问题的Wi-Fi网络管理员和安全专业人员。它适用于Wi-Fi 802.11 a / b / g / n / ac网络。它捕获每个数据包，让您看到网络的有用信息。您还可以获得有用的信息，如协议分配，接入点，信号强度等。该工具提供有关网络的关键信息，对网络管理员来说具有很高的价值。

下载CommonView：http : //www.tamos.com/products/commwifi/

Pyrit

Pyrit也是一个很好的工具，可以让您对IEEE 802.11 WPA / WPA2-PSK认证进行攻击。此工具免费提供，并托管在Google代码上。所以在未来几个月可能会消失。它适用于包括FreeBSD，MacOS X和Linux在内的各种平台。

它会执行暴力攻击来破解WPA / WPA-2密码。这是非常有效的，我建议你尝试一次。由于其有效性，有必要在此列表中提及此工具。

下载Pyrit：https : //code.google.com/p/pyrit/

最后的话

在这篇文章中，我添加了二十种免费或开源许可证的无线破解工具。您可以尝试这些工具来访问无线网络，而无需知道其密码。大多数工具能够破解无线网络密码，但密码破解时间可能会因密码的复杂性和长度而异。很少的工具不能直接用于破解无线密码，但分组分析有助于猜测密码。

我也建议使用这些工具只是为了学习目的。我们不鼓励非法活动，不支持这些人。攻击无线网络获得未经授权的访问是网络犯罪。所以，不要把自己置于危险之中。

如果您进入网络安全行业，您必须了解这些工具。

我尽力提供大多数可用的无线黑客工具。如果您有任何建议，您可以在下面填写评论或建议给我们。
