FAME：一个的开源恶意软件分析框架

Jarry • 2017年3月23日 • 工具 • 阅读 449

项目地址：

https://github.com/certsocietegenerale/fame

简介

FAME是首字母缩略词，意思是“FAME Automals Malware Evaluation”。它旨在促进对恶意文件的分析，尽可能加速和自动化完成对恶意软件的分析。

最好的环境下：分析师放样，等待几分钟，FAME能够确定恶意软件系列并提取其配置和IOC。

FAME应该被看作是一个框架，将为您的恶意软件分析开发工作提供支持。

安装

本页介绍如何在Ubuntu 16.04上安装FAME。FAME用Python编写，您可以将其安装在您选择的系统上。

安装依赖

$ sudo apt-get install git python-pip python-dev
$ sudo pip install virtualenv

MongoDB

FAME也使用MongoDB作为数据库。您不应该使用apt安装MongoDB，因为使用它的旧版本将无法正常工作。请遵循MongoDB网站上提供的安装指南：https://docs.mongodb.org/manual/tutorial/install-mongodb-on-ubuntu/

MongoDB也可以不与FAME安装在同一服务器

如果需要，您应该确保在系统启动时使MongoDB启动：

$ sudo systemctl star mongod

确保MongoDB已启动：

$ sudo systemctl star mongod

默认情况下，MongoDB只侦听localhost。如果您的MongoDB与FAME在不同的服务器上，或者您打算使用远程工作，则应通过注释bindIp指令来更改配置文件（/etc/mongod.conf）中的此设置：

net:
  port: 27017
#  bindIp: 127.0.0.1

建议在MongoDB服务器上启用身份验证。如果要做这个，首先创建管理员用户以及FAME的用户：

$ mongo
> use admin
switched to db admin
> db.createUser({ user: "admin", pwd: "SOME_STRONG_PASSWORD", roles: [ { role: "userAdminAnyDatabase", db: "admin" } ] })
Successfully added user: {
   "user" : "admin",
   "roles" : [
       {
           "role" : "userAdminAnyDatabase",
           "db" : "admin"
       }
   ]
}
> use fame
switched to db fame
> db.createUser({ user: "fame", pwd: "SOME_OTHER_STRONG_PASSWORD", roles: [ { role: "dbOwner", db: "fame" } ] })
Successfully added user: {
    "user" : "fame",
    "roles" : [
        {
            "role" : "dbOwner",
            "db" : "fame"
        }
    ]
}

然后，您必须在配置文件（/etc/mongod.conf）中启用RBAC ：

security:
  authorization: enabled

这些修改只有在mongod重启后才有效：

$ sudo systemctl restart mongod

安装FAME

首先克隆资源库：

$ git clone https://github.com/certsocietegenerale/fame
$ cd FAME

运行安装脚本，并为安装类型选择’1’：

$ utils / run.sh utils / install.py

运行FAME

为了使FAME正常工作，您应该运行：

$ utils / run.sh webserver.py

以及worker：

$ utils / run.sh worker.py

安装uwsgi：

$ sudo pip install uwsgi

创建用于Web服务器systemd配置文件，在/etc/systemd/system/fame_web.service：

[Unit]
Description=FAME web server

[Service]
Type=simple
ExecStart=/bin/bash -c "cd /REPLACE/WITH/YOUR/PATH/fame && uwsgi -H /REPLACE/WITH/YOUR/PATH/fame/env --uid REPLACE_WITH_YOUR_USER --gid REPLACE_WITH_YOUR_USER --socket /tmp/fame.sock --chmod-socket=660 --chown-socket REPLACE_WITH_YOUR_USER:www-data -w webserver --callable app"

[Install]
WantedBy=multi-user.target

为worker创建第二个systemd配置文件，在/etc/systemd/system/fame_worker.service：

[Unit]
Description=FAME workers

[Service]
Type=simple
User=REPLACE_WITH_YOUR_USER
ExecStart=/bin/bash -c 'cd /REPLACE/WITH/YOUR/PATH/fame && utils/run.sh worker.py'

[Install]
WantedBy=multi-user.target

启动这两个服务

$ sudo systemctl enable fame_web
$ sudo systemctl enable fame_worker
$ sudo systemctl start fame_web
$ sudo systemctl start fame_worker

使用nginx

安装nginx：

$ sudo apt-get install nginx

删除默认配置文件：

$ sudo rm /etc/nginx/sites-enabled/default

使用以下内容创建文件/etc/nginx/sites-available/fame：

upstream fame {
    server unix:///tmp/fame.sock;
}

server {
    listen 80 default_server;

    # Allows big file upload
    client_max_body_size 0;

    location / {
      include uwsgi_params;
      uwsgi_pass fame;
    }

    location /static/ {
      alias /REPLACE/WITH/YOUR/PATH/fame/web/static/;
    }
}

启用您的配置文件，并重新启动nginx：

$ sudo ln -s /etc/nginx/sites-available/fame /etc/nginx/sites-enabled/fame
$ sudo systemctl restart nginx

访问FAME

如果您按照说明在生产中安装FAME，现在应该可以访问http://localhost/FAME 。

否则，开发版本应该可以在http://localhost:4200/。

您现在可以按照“ 管理员指南”进行操作。

安装远程worker

FAME可以拥有多个工作用户。这可能比较有用，以便在同一时间分析更多的恶意软件，或具有不同的功能（例如，Windows工作人员可以使用不同的工具）。

远程worker的安装过程相同，步骤较少。如果已经安装了工作的FAME，则只能添加一个远程工作程序。

安装依赖关系：

$ sudo apt-get install git python-pip
$ sudo pip install virtualenv

克隆存储库：

$ git clone https://github.com/certsocietegenerale/fame
$ cd fame

运行安装脚本，并为安装类型选择’2’：

$ utils/run.sh utils/install.py

现在你可以开始使用worker

$ utils / run.sh utils / worker.py

在Windows上安装

在Windows上安装时，请安装以下依赖项：

  • Python 2.7（https://www.python.org/）
  • Git（https://git-scm.com/download/win）

然后，您可以按照相同的安装说明进行操作：

> pip install virtualenv
> git clone https://github.com/certsocietegenerale/fame
> cd fame
> utils\run.cmd utils\install.py

在启动FAME之前，请确保在Windows上遵循python-magic的具体安装说明（https://github.com/ahupp/python-magic#dependencies）。

更新FAME

使用以下命令以更新

$ utils/run.sh utils/update.py

然后，不要忘记重新启动webserver和worker，以使更改生效。

$ sudo systemctl restart fame_web
$ sudo systemctl restart fame_worker
