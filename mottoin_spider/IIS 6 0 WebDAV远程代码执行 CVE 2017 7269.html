IIS 6.0 WebDAV远程代码执行（CVE-2017-7269）

Moto • 2017年3月28日 • 漏洞预警 • 阅读 714

漏洞概述

开启WebDAV服务的IIS 6.0存在缓存区溢出漏洞导致远程代码执行，目前针对 Windows Server 2003 R2 可以利用，该漏洞最早在2016年7,8月份开始在被利用。

漏洞详情

  • 漏洞编号：CVE-2017-7269

在Microsoft Windows Server 2003 R2里的Internet Information Services（IIS）6.0的WebDAV服务中的ScStoragePathFromUrl函数存在缓冲区溢出，允许攻击者通过以“If：<http：//”开始长header头的PROPFIND请求执行任意代码。

影响范围

  • 风险等级：高风险
  • 漏洞风险：攻击者通过利用漏洞可以实现远程代码执行
  • 影响版本：Microsoft Windows Server 2003 R2开启WebDAV服务的IIS6.0

漏洞分析

漏洞关键点

  1. ScStoragePathFromUrl函数被调用了两次
  2. 制作了PROPFIND数据
