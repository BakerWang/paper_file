非常好用的WordPress安全插件

Sam • 2016年12月30日 • Web安全 • 阅读 1937

作为全球最受欢迎的博客平台之一，WordPress拥有数百万用户群。然而人气也是一把双刃剑， 一个网站越受欢迎，收到垃圾邮件或受到黑客攻击的机会越高。与其他在线博客不同，这是WordPress需要面对的问题，虽然它有强大的安全功能。因此，你的首要任务应该是确定自己博客的安全，并阻止黑客窃取和更改你的私人信息。

下面将为师傅们介绍提升网站安全性的15款WordPress插件。

1. WP Security Audit Log

这个插件记录一切发生在你的WordPress博客或网站的日志。你可以轻松跟踪任何可疑活动，并确定威胁级别。发生以下情况下将接收到实时的安全警报：

  • 创建新的用户配置文件
  • WordPress升级
  • 用户安装并激活WP主题
  • 更改其电子邮件地址或密码
  • 修改现有页面、帖子或类别
  • 登录尝试失败
  • 其他……

2.Security Ninja

结合WP安全最佳实践的一个插件，Security Ninja执行超过31个安全测试，如暴力攻击。彻底检查网站是否存在漏洞和安全漏洞，并采取预防措施来阻止攻击。

3.BulletProof Security Pro

当您安装此插件后，单击即可保护你的网站根目录文件夹和’wp-admin’文件夹。 不再需要担心如XSS、CSRF、SQL注入、Base64、RFI和代码注入等黑客攻击。激活垃圾邮件和黑客保护功能需要付费。

4.iThemes Security

作为一个新手，你是否需要寻找一个操作简单并且功能强大的安全插件。iThemes Security修复WordPress博客脆弱性问题，提高安全性。插件仪表板包含操作项目清单，按重要性降序排列，您可以根据自己的需求打开或关闭。

5.6Scan Security

6Scan为网站提供基于规则的保护，并定期更新安全功能。 内置的安全扫描器检查并保护您的网站免受各种威胁，但这个插件的最重要的功能是自动修复漏洞。任何易受攻击的代码都使用自动修复服务器端代理解决方案自动修复。此外，恶意软件相关的问题很容易受到6Scan的关注。

6.All in One WPSecurity

All in One WPSecurity插件功能构成：

  • 黑名单功能
  • 系统文件安全
  • 用户登录安全
  • 定期插件更新
  • 数据库安全
  • 用户帐户安全
  • 防火墙设置
  • 简单的用户界面

All in One WPSecurity和Firewall包含所有这些功能，这些功能使得其成为可用的最有效的WordPress安全插件之一。

7.Sucuri Security

作为最好的网络安全插件之一，Sucuri Security是一个非常全面的工具，跟踪网站上的活动、容易被人忽略的安全漏洞，用户很容易查看活动日志并注意到异常情况。

8.BruteProtect

暴力攻击是黑客攻击最常用的方法。但是使用BruteProtect插件后，你将不必在为此担心。每当特定的IP地址尝试登录到你的网站并失败的次数比平常更多，它会自动被BruteProtect插件阻止。

9.AntiVirus for WordPress

这个插件帮助你的WordPress博客反恶意软件、病毒和蠕虫，并发送有关可能的攻击的警告。这个插件的亮点是支持多语言，这个功能并不是所有插件都具备的。

10.Acunetix WP Security

一个免费但强大的安全插件，以保护你的WP，Acunetix就是解决方案。 它提出纠正措施建议，以提高您的数据库的安全性，安全文件权限，隐藏版本，并为您的博客提供管理保护。系统会检查您的网站是否存在安全漏洞，如果发现任何漏洞，立即建议纠正措施。

11.WP-DBManager

WP-DBManager插件能够优化网站的数据库、正确地修复、备份数据库，在必要时恢复、删除备份、轻松地清空/删除表，并运行特定的查询。 此插件甚至支持对数据库的优化，修复和备份任务的自动调度。

12.VaultPress

这个插件提供实时备份，网站的内容将实时同步自动的安全扫描，存在漏洞威胁，插件会立即处理。

13.Clef Two-Factor Authentication

对于那些不想处理密码或只是想要一个更好、更安全的登录过程中的人，这个插件是完美的解决方案。 打开手机上的Clef应用程序，然后将其保存在WordPress登录屏幕前面。您的下一个任务是在两个设备上排列模式。只有当他们检测到你的WP网站时，你才能登录。

14.Wordfence Security

这个免费的安全和性能插件可以为你的WordPress网站加速进50倍，你的网站上已有的任何感染将被彻底检查，使其更安全。虽然这个插件是免费的，你可以使用高级版本附带的高级功能，如country blocking、双因素短信认证服务，并能够按照你的需求安排扫描。

15.Google Authenticator

双因素身份验证是登录你的网站时的方式，而此插件提供了这一点。你再也不必担心黑客可以访问你的网站并发现你的密码。 你必须在智能手机上下载Google Authenticator应用才能使用。

对网站和博客的恶意攻击变得越来越普遍，网站所有者采取适当的预防措施保卫自己的博客是必要的。 对于不知道如何编码的用户，插件是一个最好的选择，有很多WP插件提供强大的安全功能。，你可以在这些插件中选择一个适合你的。
