ProxyDroid：Android全局代理应用

Moto • 2016年9月25日 • 工具, 技术控 • 阅读 2742

项目主页

https://github.com/madeye/proxydroid

谷歌商店 https://play.google.com/store/apps/details?id=org.proxydroid

简介

ProxyDroid 是 Android 上的一个全局代理应用，可以帮助你设置 Android 设备上的代理。

安装依赖

  • JDK 1.6+
  • Maven 3.0.5
  • Android SDK r17+
  • Android NDK r8+
  • Local Maven Dependencies

使用Maven Android SDK部署安装所有Android相关的依赖：

git clone https://github.com/mosabua/maven-android-sdk-deployer.git 
pushd maven-android-sdk-deployer
export ANDROID_HOME=/path/to/android/sdk
mvn install -P 4.1
popd

BUILD

调用building

mvn clean install

使用

一、对ProxyDroid进行配置（基本配置：）

  1. Auto Setting不勾选，我们手动进行配置。
  2. Host：输入代理服务器IP。
  3. Port：输入代理服务器端口。（HTTP默认808，SOCKS默认1080，具体视服务器情况而定）
  4. Proxy Type选择代理服务器提供服务类型：我这里选择Socks5。
  5. Auto Connect为当2G/3G/WIFI网络开启时，自动开启代理服务。不勾选，我们手动启动，以获取最大灵活性。
  6. Bypass Addresses：相当于黑名单列表，选择排除代理的IP范围，有需要的可以自己手动设置。

二、认证信息配置：

  1. Enable Authentication：如果代理服务器需要账户、密码认证，勾选。
  2. User：认证账户名。
  3. Password：认证密码。
  4. NTLM Authentication：NTLM/ NTLM2，Windows早期的一种认证方式，不用勾选。

三、特征设置：

  1. Global Proxy：一定要勾选，即为全局代理，代理所有App。
  2. Individual Proxy：单独代理所选App，勾选了（1）的不用管。
  3. Bypass Mode：勾选了代表（2）中所选App不代理，勾选了（1）的不用管。
  4. DNS Proxy：开启DNS代理。

四、通知设置：

  • Ringtone：选择通知铃声。
  • Vibrate：选择连接发生变化时是否震动提醒。
