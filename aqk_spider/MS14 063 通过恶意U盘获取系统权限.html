MS14-063 通过恶意U盘获取系统权限

2014-11-03 16:54:19 阅读：6436次 点赞(0) 收藏 来源： Icewall



上周微软发布了关于FAT32磁盘分区驱动程序中的漏洞的补丁：

MS14-063 – Vulnerability in FAT32 Disk Partition Driver Could Allow Elevation of Privilege (2998579)[CVE-2014-4115]

下面我们来看看这份报告一些有意思的地方。

微软报告中说下面的系统版本受到影响：

Microsoft Windows Server 2003 SP2

Vista SP2

Server 2008 SP2

但是，你懂得。。。微软今年停止维护的XP也会受到这个漏洞的影响。

有视频有真相，下面这个视频展示了这个漏洞是怎么影响Windows XP SP3的。

漏洞存在在FastFAT.sys 驱动中，通过u盘传递恶意的FAT32 Boot sector 被利用。
