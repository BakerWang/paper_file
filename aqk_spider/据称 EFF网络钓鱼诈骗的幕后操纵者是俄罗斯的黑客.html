据称，EFF网络钓鱼诈骗的幕后操纵者是俄罗斯的黑客

2015-08-31 12:38:10 阅读：14013次 点赞(0) 收藏 来源： 安全客
作者：lazynms

EFF组织的安全研究专家推测，此次最新的EFF网络钓鱼诈骗事件的幕后操纵者就是由俄罗斯所资助的APT 28黑客组织成员。

据称，这个APT组织有俄罗斯的国家资助，该组织的黑客在这个月注册了一个虚假的电子前哨基金会域名。黑客在注册域名时，使用了这个目前十分热门的隐私看门狗的名字，并以此来欺骗目标用户，尝试让目标用户误以为这封电子邮件（实际上是网络钓鱼邮件）是由一个受信任的组织所发送的。

目前已经有数名安全研究专家正在对此次事件进行调查，根据他们的描述，此次网络钓鱼活动是Pawn Storm计划的一部分。这些安全研究专家来自于全球各大安全公司，他们推测，俄罗斯的APT 28组织就是这个长期的网络间谍活动背后的始作俑者。

Pawn Storm组织，也被称为APT 28，该组织已经活跃了数年之久了。FireEye公司的安全研究专家已经对该组织的攻击活动进行了长期的跟踪和分析，他们表示，该组织的攻击活动都是在莫斯科时间的工作时间进行的，并且攻击的目标都是一些与政府，军队和安全组织有关的信息管理机构。除此之外，研究专家还认为，该组织黑客所窃取到的信息对于俄罗斯情报机构来说，是十分有价值的。

根据EFF所发表的官方公告，攻击者在几周之前注册了一个伪造的EFF域名（electronicfrontierfoundation.org），并且使用这个虚假域名和一个近期才修复的Java 0 day漏洞来对目标用户实施攻击。

尽管Oracle公司已经修复了这个漏洞了，但是该黑客组织仍然可以利用这个漏洞来发动攻击。

在网络钓鱼邮件中，包含有一条虚假EFF网站的地址链接，当用户点击了这条地址链接之后，目标用户将会被重定向至一个相同域名的网页，而这个网页则会加载一个旧版本的Java applet。没错，这个版本的applet是包含Java 0 day漏洞的。

攻击过程是十分复杂的，它利用了一个近期才被发现的Java漏洞，这也是安全研究人员在两年内首次发现的一个Java 0 day漏洞。攻击者可以向目标用户发送一封带有特殊URL链接的网络钓鱼邮件，而这个URL链接指向的则是一个恶意域名（此次事件中，攻击者使用的是electronicfrontierfoundation.org）。当目标用户点击了这个URL链接之后，这个特殊的URL链接将会把访问用户重定向至另一个特殊的URL地址，该地址格式为：htt://electronicfrontierfoundation.org/url/{6_random_digits}/Go.class，这个页面包含一个存在漏洞的Java applet。当用户访问了这个页面之后，系统就会接收到Java的有效载荷，在此之后这个URL地址就会被禁用，并且再也不会发送任何的恶意软件了（这也就使得恶意软件分析人员的工作变得更加的困难了）。

现在，通过这个Java漏洞，攻击者就可以在目标用户的设备上运行任意的代码，以及下载其他的恶意有效载荷了，这些二进制恶意程序均可以在目标用户的计算机中运行。

在分析过程中，我们可以从electronicfrontierfoundation.org网站中恢复一些恶意的Java代码样本。

Go.class将会执行App.class，而在这个class文件中，则包含了实际的攻击代码。App.class的有效载荷能够利用那个Java 0 day漏洞（由趋势科技所报道的Java 0 day漏洞），然后自动将第二阶段攻击所需要的恶意二进制代码（即cormac.mcr）下载至用户的home目录下，并随机选择一个字符串为这个文件进行命名，并且还会将文件的后缀名改为”.exe”。其中最有趣的地方就是，App.class中的代码还可以下载一个*nix二进制代码，这也就意味着，这种攻击很有可能会对Mac或Linux平台的用户造成影响。

EFF的安全研究专家们表示，此次攻击事件与俄罗斯APT 28组织其他的攻击事件相比，有非常多的共同点。比如说，在这些攻击事件中，攻击者所使用的恶意文件的文件名和文件路径是一样的，除此之外，相同的还有Java有效载荷以及Java漏洞。攻击者还使用了Sednit（一种恶意软件）来感染目标用户的计算机系统，并安装后门程序。在Windows平台上，恶意软件会将Sednit下载至当前登录用户的home目录下，并使用一个随机生成的字符串作为文件名，然后执行这个恶意文件。恶意payload不仅会与一系列的系统服务挂钩，而且还会下载一个DLL文件。系统会执行这个DLL文件，然后与一个由攻击者控制的服务器进行连接，它还会记录用户的键盘输入。除此之外，它还能够根据攻击者的命令请求来在目标系统中执行相应的攻击模块。

在过去，该组织还发动过很多针对俄罗斯持反对意见的政治家和记者，美国国防设施承包商，北约部队，以及白宫工作人员的攻击事件。

此次事件应该让所有人都敲响警钟，每个人都应该有防范网络钓鱼攻击的安全意识。我们给广大用户提供了一些能够帮助你们提升安全性的建议，大家应该小心恶意邮件，尽量避免遭受此类网络钓鱼攻击。
