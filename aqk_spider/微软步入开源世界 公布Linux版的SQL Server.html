微软步入开源世界，公布Linux版的SQL Server

2016-03-08 13:22:24 阅读：29140次 点赞(0) 收藏 来源： 安全客
作者：WisFree

SQL Server是微软公司推出的一款关系型数据库管理系统，它具有使用方便，可伸缩性好，与相关软件集成程度高等优点。SQL Server是一个全面的数据库平台，使用集成的商业智能 (BI)工具，并为用户提供企业级的数据管理服务。微软SQL Server 数据库引擎为关系型数据和结构化数据提供了更安全可靠的存储功能，使用户可以构建和管理用于业务处理的高可用和高性能的数据应用程序。

这一年对于微软公司的数据业务以及整个数据处理行业而言，都是令人难以置信的一年。在本周周四时间，微软公司将会在位于美国纽约的数据驱动中心正式向全世界推出SQL Server 2016，但是用户仍然需要等到今年年底才能获取到相关的产品。此次发布会可以算得上是SQL Server历史上最重要的一次产品发布会了，我们将会在这款新产品中添加大量的新技术以及实用性很高的新功能。SQL Server 2016将会提供下列功能：

l   引入开创性的安全加密保护功能，可以保证存储在服务器中的数据都是经过加密处理的，并且可以为传输过程中的数据提供最大程度的安全保护；

l   服务器数据库能够给每一个独立的工作负载提供优质的数据服务，系统的数据处理和工作性能得到了大幅度的提升；

l   数据仓库性能得到大幅度提升；

l   为每一名员工和每一台设备提供数据情报服务，支持的设备包括新款苹果手机，安卓手机，以及Windows Phone手机等等；

l   提供高级的数据分析服务，客户可以对业务操作以及业务数据进行实时分析与预测；

l   独特的云功能，允许用户在云端部署混合数据架构，支持分区处理数据和工作负载。客户可以在住所以及其他联网的地方随时随地与云端服务器同步数据，为客户节约成本，提高工作的灵活性。

SQL Server将会给用户带来上述所有的新功能以及服务改进，它不仅仅只是一个新型的数据库系统，而且它还是一个可以用于数据管理，业务分析和智能处理的完整数据平台。无论是在本地还是在云端，该平台还可以保证用户数据的一致性。实际上，在过去的一年中，我们已经在超过一百四十万个SQL云数据库中使用了SQL Server 2016的核心代码，并且它还为Azure SQL数据库提供数据处理服务，所以它已经在现实世界的实践操作中积累了丰富的经验，而这些经验造就了SQL Server 2016的强大，也造就了这样的一款身经百战的数据处理平台。

根据Gartner近期最新的报告显示，Gartner公司认为微软是目前数据处理行业中的领头羊。

我们对SQL Server进行扩展，现在它也可以在Linux操作系统上运行了

今天，我很高兴地宣布，微软公司将会把SQL Server带到Linux操作系统之中，从此以后Linux也可以使用SQL Server了。SQL Server将会成为一个真正跨平台的数据平台，它将会给Windows Server以及Linux用户提供高度一致性的数据处理功能。Linux版的SQL Server同样也会给用户提供本地和云端的混合服务。今天，我们将会给大家展示新版SQL Server数据库的核心功能。

Linux版的SQL Server可以为用户的数据解决方案提供更大的灵活性。执行关键数据处理业务时的性能，业界领先的TCO，一流的数据安全保障服务，以及混合云创新服务（例如Stretch数据库，该数据库允许客户在任何地方都可以随时随地访问数据库中的数据），新版SQL Server已经将上述所有的这些先进功能全部都集于一身了。

新版SQL Server能够将其著名的高可信度数据库服务提供给更多的用户，对于微软公司而言，这将是一个非常重要的决定。在此之前，微软公司曾经向客户承诺过，公司将会朝着跨平台数据服务解决方案提供商的方向去发展，而现在公司将这一关键产品带到了Linux操作系统之中，这也是微软公司朝着目标方向发展所迈出的重要一步。这将给用户提供更多的选择，用户也不必再担心由操作系统差别所带来的不便性了。微软公司也期待这样的改变能够提升SQL Server的市场占有率。

红帽公司的产品技术研发主任Paul Cormier认为：“新版SQL Server能够为全世界的Linux企业级用户提供非常宝贵的企业数据处理经验和数据分析能力，这对于用户而言，的确是一笔宝贵的财富。我们相信用户在得知这一消息之后，将会非常的开心，而且也非常愿意看到微软公司进一步将其产品引入至Linux操作系统之中。由于我们目前正在深度构建混合云服务，涉及到的不仅是Linux，而且还有大量中间件和PaaS平台。红帽企业版Linux很高兴能够与SQL Server进行深度合作，并给广大企业级用户提供更多的数据库选择。”

Cononical公司的创始人Mark Shuttleworth表示：“我们很高兴能够与微软公司合作，感谢微软将SQL Server引入了Linux。目前，大多数的客户已经在Ubuntu中使用Azure Data Lake服务了。现在，开发人员就可以使用SQL Server来开发出更加高效的数据处理程序，并为广大的企业用户提供更加高效的企业级数据处理服务器。”

我们发布SQL Server的Linux版本，是为了满足更广大用户的实际需求，而且这也是公司的一种创新方式。

就在上周，微软公司正式宣布收购热门初创企业Xamarin。近期，我们还发布了Microsoft R Server，这款产品给用户提供了开创性的数据分析服务，并且还支持Hadoop和Teradata。去年的一月份，微软公司宣布了收购Revolution Analytics的计划。Revolution Analytics是一家针对统计学计算和预测分析分发的R编程语言工具提供商。在去年四月份，微软公司完成了这项收购，同时宣布将在自家的SQL Server中集成R语言，随后便发布了Microsoft R Server。但是微软公司表示，在SQL Server 2016正式发布之前，支持Windows平台的Microsoft R Enterprise仍然是一个独立的产品。另外，微软公司开放了新版本的R Server开发者版本（Microsoft R Server Developer Edition），并在产品中加入了商务级数据处理功能，开发者可以免费下载和使用。

从今天开始，用户可以下载和安装Linux版SQL Server的个人预览版，我们希望能够与开源社区的广大开发者进行合作，并在客户以及合作伙伴的帮助下将这款新版的SQL Server推向市场。
