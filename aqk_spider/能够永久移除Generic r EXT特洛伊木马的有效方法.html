能够永久移除Generic.r_EXT特洛伊木马的有效方法

2015-08-31 23:59:23 阅读：12062次 点赞(0) 收藏 来源： 360安全播报

Generic.r_EXT将会攻击你的计算机系统,但是你却不知道如何清除这种病毒?那么请你赶紧阅读这篇文章吧,我们将会给大家介绍一些有关Generic.r_EXT木马的知识,并且将会告诉大家如何将这种木马病毒从你们的计算机中清除掉。

什么是Generic.r_EXT?

Generic.r_EXT是一种特洛伊木马病毒,这种病毒可以进行一系列破坏性的活动,然后拖垮你计算机的操作系统。与此同时,还会引起一些其他的计算机问题,甚至有可能会损坏你的硬件设备。通常情况下,Generic.r_EXT可以通过垃圾邮件的附件,不安全网站的种子文件,或者捆绑在免费软件上来进行传播。如果你的计算机处于一种不安全的环境之中,那么这种病毒一旦抓住机会,就会毫不犹豫地进入你的系统。

当Generic.r_EXT进入了你的计算机之后,它将会修改重要的系统设置,为了让设置立即生效,它甚至还会修改注册表中的相关信息。Generic.r_EXT还允许攻击者在目标系统中留下后门。这一操作可以由攻击者执行,也可以通过病毒来在目标系统中自动执行。除此之外,像Generic.r_EXT这样的特洛伊木马可以窃取目标主机中的数据,这些数据包括机密文件,财务信息,用户密码以及支付卡信息等等。

更加糟糕的是,它还可以进入你的通讯录,并且在你完全不知情的情况下,用你的名字来给你的朋友们发送带有木马病毒附件的电子邮件。然后病毒就会感染你朋友的电脑。因此,一旦你在你的计算机中检测到了Generic.r_EXT的话,你最好赶紧在它还没有拖垮你的计算机之前,将它从你的电脑中移除。

请大家按照我们在下列给出的指导教程来手动删除这种木马病毒,或者使用功能强大的恶意软件移除工具-Anvi Smart Defender来快速移除Generic.r_EXT,而且这款病毒防御软件还可以为你抵御更多的恶意软件威胁。

怎样摆脱Generic.r_EXT?

选择一,手动清除Generic.r_EXT

-第一步:重启你的计算机系统,并进入带有网络连接的安全模式

-第二步:结束掉可疑的进程

-第三步:显示隐藏的文件和文件夹,并将与这个特洛伊木马有关的文件全部删除

-第四步:使用注册表编辑器删除注册表中所有的相关项

选择二,自动清除Generic.r_EXT

-第一步:下载并运行Anvi Smart Defender

-第二步:使用Cloud System Booster来优化你的Windows系统

请注意:手动移除该木马的过程是十分复杂的,只建议专业的计算机用户进行操作。如果你并不是计算机方面的专业人士,或者无法通过手动方法解决这一问题的话,我们建议用户使用Anvi Smart Defender,这款防御软件可以帮助你轻松快捷地移除系统中的恶意软件。除此之外,Anvi Smart Defender可以帮助你检测和移除恶意软件所带来的威胁,还可以实时保护你的计算机系统免受目前增长迅速的病毒威胁,例如木马,rootkit,间谍软件,流氓软件等等。与此同时,欢迎大家下载免费的系统清理和优化工具-Cloud System Booster。

选择一,手动清除Generic.r_EXT

第一步:重启你的计算机系统,并进入带有网络连接的安全模式

-->重启你的计算机,在整个系统启动的过程中,按住F8,直到高级启动选项出现在你的屏幕上。使用小键盘的上下键将光标移动到“Safe Mode with Networking”选项,并按下Ender键选择该模式。

第二步:结束掉可疑的进程

-->打开任务管理器(或者同时按下Ctrl+Alt+Del)。

-->点击进程列表。

-->找到与Generic.r_EXT有关的进程,选中相关进程并点击“结束进程”。

第三步:显示隐藏的文件和文件夹,并将与这个特洛伊木马有关的文件全部删除

-->点击开始菜单,在搜索框中输入“文件夹选项(folder options)”,并在搜索结果中点击“文件夹选项(Folder Option)”。

-->在文件夹选项中,点击“查看(View)”选项,在“高级设置”中,选中“显示隐藏的文件,文件夹和驱动器”选项,取消勾选“隐藏受保护的操作系统文件(建议)”选项,完成所有设置后,请点击“确定”按钮。

第四步:使用注册表编辑器删除注册表中所有的相关项

-->点击开始菜单,在搜索框中输入“regedit”,然后在搜索结果中点击“regedit.exe”。

-->在注册表编辑器中,搜索所有与该木马病毒有关的注册表键,然后将它们从注册表中删除掉。

选择二,自动清除Generic.r_EXT

第一步:下载并运行Anvi Smart Defender

Anvi Smart Defender v2.3是一款功能强大的恶意软件检测工具,它可以检测并移除恶意软件,实时保护你计算机系统的安全。这个版本使用了双重扫描引擎(本地引擎和云引擎),因此可以有效地检测病毒,可以检测的病毒包括木马,rootkit,间谍软件,流氓软件等等。除此之外,这个版本的反病毒软件还可以实时检测计算机的系统服务,启动项,恶意软件和内存数据等等,但无法进行磁盘扫描。现在,不同的安全威胁正在迅速发展和增长,但功能强大的反病毒软件可以帮助你保护计算机系统的安全,陪你一同面临这一巨大的安全挑战。

下载地址:

http://www.anvisoft.com/software/asd/?refer=resources-roy

1. 安装并运行Anvi Smart Defender,然后点击“扫描“选项。

2. 选择“快速扫描“或”完整扫描“,在扫描结束之后,Anvi Smart Defender将会把扫描结果提供给用户,此时请点击“应用”按钮。

第二步:使用Cloud System Booster来优化你的Windows系统

Cloud System Booster是一款系统清理和优化工具,它可以为Windows的用户提供系统维护解决方案。它能够保护你的在线隐私,使你的Windows计算机运行得更加流畅以及更加安全。这款软件非常的小巧,不但易于使用,而且还可以给用户提供综合的系统清理以及系统优化功能。

下载地址:

http://www.anvisoft.com/software/csb/?refer=resources-roy

1. 下载并运行Cloud System Booster。

2. 在软件安装完成之后,请运行它,然后点击软件主界面的“快速清理”按钮来进行“快速扫描“或者”完整扫描“。

3. 当软件扫描完成之后,它会给用户提供一个扫描结果的界面,请单击“清理”按钮,点击之后软件将会把系统中的垃圾文件和无效的注册表键等系统垃圾全部清除掉,这样就可以保证你的计算机系统是干净的。

请注意:为了降低病毒再次进入计算机系统的可能性,请按照以下步骤来清理你浏览器中的cookie。

4. 回到Cloud System Booster的主界面->点击“磁盘扫描”选项->拉动滚动条,并选择“Internet Explorer Cookies”,“ Firefox Cookies”和“Google Chrome Cookies”->点击“开始清理”选项。
