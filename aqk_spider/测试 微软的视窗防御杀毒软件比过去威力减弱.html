​测试：微软的视窗防御杀毒软件比过去威力减弱

2016-01-28 18:10:41 阅读：23237次 点赞(0) 收藏 来源： 安全客
作者：默白

事实上，它比其他付费的同等软件排名更靠前

我们常常认为微软免费的视窗杀毒软件很讨厌，基线第三方公测已经被认为很平常。但是，最近有独立测试显示，它事实上胜过一些第三方程序组，这些程序组是需要付费的。

由AV-test.org12月公布的结果显示，对于家庭用户来说微软在反病毒程序组的选择中位于中间位置。微软仍然接近市场众多企业的底部，然而，微软系统中心的版本已经将Windows 10的版本完善了。

微软在12月AV-test.org.的测试中攀升到了反病毒程序组的中间位置。

这个结果是有效的。在2013年12月，AV-test.org测试了23个反病毒程序供应商应对现实世界中的抽样恶意软件，微软利用Windows 8.1完成了测试。2015年12月，同样是利用Windows 8.1，微软在19个竞争者中获得了第11名超过了 F-Secure、ESET、Panda Software 和Comodo.这些公司。

这为什么很重要：如果你想要一个免费的杀毒软件，大多数供应商为您提供了选择。但是并不是所有供应商都会这样做：例如， F-Se，它的反病毒程序解决方案的每个策略每年会收取30磅（42.74美元）。没有免费的选项。（这个公司不通过网页提供免费的请求式反病毒程序扫描仪，这些扫描仪能够探测到你系统里存在的恶意软件），换句话来说，你可能正在为那些不是很好的反病毒程序付款。

微软仍然在为保护你的个人电脑努力

根据AV-test.org的调查结果，微软在捕捉11月和12月较为常见的恶意软件方面十分有效。研究99.6%在这两个月展开。但是微软的防火墙只在11月和12月分别捕捉到了97.5%和90%的测试公司抛出的所谓的零日恶意软件。

这不是最好的最顶级的反恶意软件解决方法，例如，Avira, 捕捉到了99.9%到100%的全部测试恶意软件，无论是零日代码还是著名的virus代码。（Avira、Bitdefender,和 Kaspersky 取得了最佳成绩）甚至是F-Secure，AV-test.org 排位在微软之下的软件实际上也强于微软，它捕捉了100%的零日代码和99.8%到99.9%的流行恶意软件。

那么，为什么微软排名这么靠前呢？因为微软的防火墙很少困扰你。当防火墙突然出现警告时，你可以确定这是值得警觉的。 Comodo 和 F-Secure在有些情况下错误的将一些合法软件当作恶意软件并且错误的阻挡合法软件正常发挥功能。AV-test.org 为合用性和防御性的表现都加分。在之后的目录中，微软仍然排在倒数第三。但是排在微软之后的恶反恶意软件程序也错误报告恶意软件，并且因此受到处罚。

看看 Windows 10 别的防御

AV-test.org 同样测试了在Windows 10系统64位企业版微软系统中心解决方案。在这里答案似乎更加清晰：选择一个第三方解决方案。只有Cylance’s Protect和 Seqrite’s Endpoint Security在11个供应商的角逐中低于微软的排名，根据所有防御的表现，只有 Seqrite 系统中心运作较差。

据AV-test.org在2015年12月的测试指出，BitDefender Endpoint Security 事实上是完美的，捕捉了100%的恶意软件并且只有一次误将合法软件当作恶意软件的情况，微软仍然在商业竞争中努力挣扎。

微软的视窗防御不是很好，根据防御，你甚至会认为它不好。但是，它至少它是被关注的，它也正在进步。我们仍然建议你安装一个第三方反恶意程序解决方案。但是，正如他们所说的，兴起的浪托起了所有的船只。随着微软改善了Windows的防火墙，第三方反恶意程序软件必须不断进步，不然将会有被超越的风险。
