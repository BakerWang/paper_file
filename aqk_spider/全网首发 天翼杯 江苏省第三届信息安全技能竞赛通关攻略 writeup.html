几乎两天整的奋斗，终于成功进入复赛，代码有点粗糙

数字游戏：

 C语言代码穷举：

#include<stdio.h>
void main()
{
 int a,b,c,d;
 for(a=0;a<=9;a++)
        for(b=0;b<=9;b++)
               for(c=0;c<=9;c++)
                      for(d=0;d<=9;d++)
                             if(a*4000+b*400+c*40+d*4==d*1000+c*100+b*10+a)
                                    printf("%d,%d,%d,%d\n",a,b,c,d);
}

运行结果：

a b c d

2 1 7 8

顺利穿越：

写程序咯。。。

初出茅庐：

很明了的告诉我们要绕过防注入系统

多次测试后发现注释号/**/和大小写翻转可以注入

User表的字段简单猜了一下

http://218.2.197.250/sqli/one.html?id=2/**/And/**/1=2/**/uNiOn/**/SeLecT/**/pass/**/from/**/user--

Pass字段里发现了flag

暗度陈仓：

题目说要权限绕过

用admin password登陆抓包，

发现post包里的数据改了没什么用

但随后有一个get包。。

把那个checkcode里的代码base64解密，发现是一串数字，在用ascii对应查询后发现是user

所以就把userid改为1 ranktype 改成admin checkcode改成 OTcgMTAwIDEwOSAxMDUgMTEwIA==

改完后发出去发现成功获得flag

兵不厌诈：

抓包分析后发现和服务器并没有太多的交互，于是转而分析本地文件

在一个JS文件里面发现了这样的一段代码

简单修改本地JS后就可以弹出

没错 flag就是那个js里面直接放出来的一串字符串。。。

问道于盲：

很明显的一个http头伪造的问题,本以为很快就可以pass结果后来才发现这题目真心坑！

抓包后在http头里面添加这个

X-Forwarded-For：

后面填伪装的IP。试了各种 127.0.0.0 localhost 还有网站IP都没有用。。

最后！！！

X-Forwarded-For: 192.168.1.1  ...返回出了flag

化险为夷：

Wireshark查看一点点拼起来

答案enable123

觅迹寻踪：

思路是爆破的 string 文件| grep -i vlan

然后一个个尝试

思路：ascii解码，然后在chrom console中运行脚本，得出base64密码 解码的bz2编码
