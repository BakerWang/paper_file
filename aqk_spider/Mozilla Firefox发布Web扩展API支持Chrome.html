Mozilla Firefox发布Web扩展API支持Chrome

2015-08-25 13:28:35 阅读：12777次 点赞(0) 收藏 来源： 安全客

Mozilla Firefox发布网页扩展API支持谷歌浏览器、欧朋浏览器的扩展插件

我们是否对这一消息感到高兴呢?让我们一起去看看!

火狐浏览器一直在思考的是如何将谷歌浏览器扩展,让它支持火狐浏览器的功能 

火狐的母公司即Mozilla基金会决定更新他们的插件和扩展的基础设施,使火狐更强大、更人性化。

即便排名第三,Firefox浏览器一直在努力达到第一位。

这个举动将在各种方式帮助火狐浏览器,如:

•集成新技术如 Electrolysis 和 Servo
•阻止广告软件和间谍软件
•缩短它需要加载附件的时间

由Mozilla带来的功能

根据Mozilla所陈述,由火狐浏览器附加组件带来的功能有:

1、引入一个强大的网络扩展API(应用编程接口)插件

2、为登录而附加和扩展

3、阻止XUL,XPCOM组件运行,并允许添加模型

4、多进程火狐浏览器和附加组件通过电解程序来执行

5、网络扩展的应用编程接口(API),这是被Chrome(谷歌浏览器)和Opera(欧朋浏览器)广泛使用的用于开发的插件,实现不同平台浏览器之间的访问。

有了这个应用编程接口(API),开发人员能够使火狐浏览器成为跨平台的浏览器,通过在火狐浏览器和谷歌浏览器上使相同的扩展可以让浏览器之间发生最小的变化。

同时,它可以在addons.mozilla.org(AMO)或通过开发者网站使用。网络扩展预览版是可以在Nightly上使用的,这是开发者的版本。

在多进程浏览器(也被称为电解版或E10S),扩展的代码将从内容不同的程序中运行,与当前的情况不同的是,在单一的操作系统中运行时将会处理的所有网页浏览器的内容信息。

加强安全性

对于安全性的深入开发也将带动其他功能的开发,火狐浏览器开发了一个带有强制性的登录功能,这样解决了恶意开发者利用他人逃脱自己责任的问题。

Mozilla采取的一个重要步骤是反对依赖于XUL,XPCOM和XBL的插件;火狐的基础技术。

Browser extensions扩展Web浏览器的功能,使它们更具互动性,从而更加人性化并且帮助保护用户的隐私,同时浏览网页。

Mozilla公司表示,“我们想着重于我们的插件开发部门,我们想与你一起设计新的API,通过移植和扩展,开发出新的新插件让火狐浏览器变得更好。”

Mozilla公司承诺他们将会及时的告知我们更多的开发成果以及告诉我们这些成果将如何更好的帮助我们。

更多有关Mozilla公司的详细细节,可以访问他们的官方博客,他们会在上面做出声明。

因此,在一开始就提出的问题的答案是:通过开发火狐浏览器的更多的功能来给用户留下深刻的印象,当然,开发者们会度过一段艰难的时光,来开发火狐浏览器更多的功能。
