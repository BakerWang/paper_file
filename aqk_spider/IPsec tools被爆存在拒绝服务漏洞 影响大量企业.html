IPsec-tools被爆存在拒绝服务漏洞，影响大量企业

2015-05-20 11:25:42 阅读：12443次 点赞(0) 收藏 来源： 安全客

IPsec-tools被爆存在拒绝服务0day漏洞，并且互联网上已经发布了可利用的程序。该漏洞是由一个空的引用造成崩溃，可造成IKE的守护进程拒绝服务，进行该攻击只需要发送两个很小的UDP包，由于很多企业的vpn链路都使用了IPsec-tools，一旦遭受攻击，会造成链路中断，影响正常的业务。

IPsec是什么？

Internet 协议安全性 (IPSec)”是一种开放标准的框架结构，通过使用加密的安全服务以确保在 Internet 协议 (IP) 网络上进行保密而安全的通讯。IPSec协议经常被用在远程接入的vpn上，被广泛用于企业单位中。

漏洞位置

if (iph1->rmconf->proposal->gssid != NULL) {

由于NetBSD, FreeBSD, Android和很多产品都使用了IPsec tools，所以花费大量精力来fuzz和审计的人肯定不少，但是偏偏都错过了这一点，漏洞发现者表示自己是幸运的。
