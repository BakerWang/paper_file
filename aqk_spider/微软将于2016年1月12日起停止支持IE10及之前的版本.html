微软将于2016年1月12日起停止支持IE10及之前的版本

2015-11-24 19:15:27 阅读：22620次 点赞(0) 收藏 来源： 猎狐安全平台

        一个必须面对的现实问题就是,老版本的IE浏览器存在安全漏洞,而老版又无法对接最新技术,所以明年的更改计划也是不得已而为。

        据日本Livedoor门户网站11月23日报道,自2016年1月12日起,IE浏览器的将会大幅度调整支持方针。


        据了解,该消息系2014年夏天由微软发布,然而,眼看马上就要到时间节点,却仍未引起大家的关注。但缺乏相关了解很有可能引发网络危险状况发生哦,现在就让我们看看有哪些注意事项吧。

        确认Windows系统和IE浏览器的版本

        自2016年1月12日起,IE浏览器将不再支持老版更新。

        据了解,Windows XP系统是在2014年4月9日开始停止更新的,距今已一年半了,不由得有些感慨,虽然距离现在正使用的Windows版本停止支持还有好一段时间,但还是忍不住做下总结。


       Windows Vista ……2017年4月11日

        Windows 7 ……2020年1月14日

        Windows 8.1 ……2023年1月10日

        距离最近的是Windows Vista,虽然一年半的时间让人感觉无需惊慌,但可不能这么掉以轻心。其实,2016年的1月16日是个十分重要的节点,因为从该日起,所有的Windows系统只支持最新版IE。也就是说,16日之后IE 9和IE 10将不会再有安全更新程序,假如您的系统还是使用该两个老版本,一旦发现漏洞则很有可能使网络安全陷入危险境地。

        为了防止以上糟糕状况发生,还是建议您尽快升级IE浏览器至最新版。

        此外,各版Windows可使用的IE浏览器最新版本如下所示:

       Windows Vista……IE9

        Windows 7……IE11

        Windows 8.1……IE11


        值得注意的是,如果将Windows 8升级为Windows 8.1的话,IE浏览器也会自动升级为IE 11版。不过,如果大家没有什么特殊理由,还是不要再使用Windows 7/8.1了,爽快的升级为Windows10吧。

        为了以防万一,在这里列举一下几组浏览器和系统的危险组合,如果您现在使用情况就属其中之一的,请不要忘记升级IE浏览器。

        Windows Vista + IE7/8

        Windows 7 + IE9/10

        Windows 8 + IE10

        接下来,让我们看看如何确认IE的版本:

        点击浏览器右上角的“工具”选项,打开菜单并选择版本信息一项即可。

        IE浏览器已是明日黄花 如果仍打算继续使用请选择最新版IE浏览器的支持方针将更改的这一消息是微软在去年夏天(2014年8月)公布的,具体情况大家可自行搜索察看,在此将更改理由总结如下:

        1、提高安全性

        2、减轻开发与检证负担

        3、对接最新技术

        一个必须面对的现实问题就是,老版本的IE浏览器存在安全漏洞,而老版又无法对接最新技术,所以明年的更改计划也是不得已而为。此外,也正是因为安全问题,Windows 10标准浏览器是Microsoft Edge,而IE浏览器只作为候补软件保留了下来。

        由此看来,IE浏览器虽已是明日黄花,但也仍有一批使用者,只是这些使用者大都是对浏览器不在意也对IE版本不感兴趣的用户。

        而这也正是IE浏览器更改支持方针消息不能被忽视的原因。

        2016年1月12日马上就要来临了,使用Windows Vista/7/8 的用户不要忘了确认自己的IE浏览器版本。
