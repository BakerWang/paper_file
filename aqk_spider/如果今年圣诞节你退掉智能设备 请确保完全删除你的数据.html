如果今年圣诞节你退掉智能设备，请确保完全删除你的数据

2015-12-07 13:11:53 阅读：16237次 点赞(0) 收藏 来源： 安全客
作者：默白

你永远不会把你的手机或平板电脑在街上给陌生人，对吗？

        我从Fry’s Electronics刚刚购买的平板电脑似乎有些不太对劲，因为开机后主屏幕显示的第一个提示竟然是漫威英雄冠军赛（一种手游）

        我已经为我们平板购买指南找到了最破旧、最糟糕的选择，作为假期不能购买的案例。我发现，大约70美元的Visual Land平板两年前被生产，它具有可见像素和一个松散的电源连接器。我打开电源，确保它能够工作，却发现它是别人用过的。

        每次我用平板刚一连接我的无线网络，一个显示用户帐户需要“帐户操作要求”的提示窗口马上就会弹出来，同时还会有相应的错误标志。用户在退回平板电脑后明显改变了自己的密码，意味着任何购买它的人（我）不能访问任何重要的个人信息。我将它复位了，我也不想我发的任何个人信息遗留在平板上。

        为什么这个很重要：你所拥有的每一个数字设备都是一个潜在的门户，有人可以利用它介入你的个人生活：访问您的电子邮件、社会化媒体、云存储和其他账户。是的，我们中的大多数人都已经内化了一些安全的做法：现在杀毒软件是一种生活方式，我们建议亲友们不要点击可疑电子邮件，而且我们都知道如果丢了手机应该怎么办。但是尽管我们如此防备，有时还是会将一些小的痕迹留在平板里，而且，一旦你将它退回，这些信息将在你的控制之外。


吸取教训

        然而，从这一切中吸取的教训，不是还是和这个平板的主人一样（将信息留下），然后退还这个平板电脑。是的，将一个电话，平板电脑返厂后，记住改变你的密码或者设置计算机可以锁定其他窥探访问您的帐户和其他个人信息的人。但是，如果你下载了电子邮件、照片、应用程序或其他数据，任何储存在本地的信息将保留给想偷窥你个人生活的人。

        那么你如何确保你的设备清除了所有数据呢？按照下列步骤。就能确保任何存储在设备上的存储备份到云或其他地方。一旦复位执行，设备上的所有数据都将永远消失。

安卓：

        进入设置菜单，通过从屏幕顶部下拉，点击“齿轮”图标，或直接点击设置图标都可以。向下滚动到“备份和重置”选项，然后单击。安全操作，并确保过程在返厂之前完成这些过程。

        请注意，这个过程手机和平板电脑都一样。


iOS:

        重置一个iPhone，ipad，或者iPod-touch，进入访问设置菜单然后选择“一般设置”菜单下重置选项中的清除所有内容和设置。如果你是使用查找我的iPhone”选项，设备会询问你的iCloud密码并关掉它。


Apple OS X:

        Macworld有极好的深入底漆功能可以安全地擦除您的驱动器。


Windows 8 或8.1系统:

        从屏幕的右边滑动，进入设置，然后点击改变用户设置。（如果你使用鼠标，指向屏幕的右上角，移动鼠标指针，点击设置，然后点击改变电脑设置）然后选择更新和恢复进行恢复，单击“删除所有”和“安装”窗口。你同样可以选择快速清理或彻底清除你的数据。如果你确定删除，可以点击彻底删除。


Windows 10系统: 

        进入设置菜单，然后点击更新与安全菜单下的恢复重置个人账户。确保您选择删除所有的选项，包括您的个人文件，（你可能不需要使用注销指令覆盖数据驱动器，但如果你真的要处理的PC，这是个好办法。）


Windows 系统的手机:

        点击进入“设置”菜单，然后点击“关于”，你会看到一个选项能够重置你的手机。

        一些设备还提供远程删除的选项，或使用本机操作系统或第三方应用程序删除数据。但这是自己执行装置重置更令人放心的方法。

        注： Fry's Electronics是全美超级家电连锁店。
