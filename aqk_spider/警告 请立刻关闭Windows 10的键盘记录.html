警告！请立刻关闭Windows 10的键盘记录

2015-09-09 12:44:30 阅读：28172次 点赞(0) 收藏 来源： lazynms@360安全播报
作者：lazynms

Windows 10 是美国微软公司所研发的新一代跨平台及设备应用的操作系统。

你知道吗?用户在使用最新版的Windows 10操作系统时,微软公司可以跟踪用户所输入的每一个字符,以及用户与语音助手Cortana所说的每一句话。

去年秋天,我们曾报道过,微软公司公开地将“键盘记录器”加入了Windows 10操作系统的技术预览版本中。而且微软公司还表示,他们可能会收集用户的语音信息以及通过外部交互设备所输入的字符。

很多技术人员还以为微软公司只会在Windows 10的技术预览版中添加键盘记录器,而且目的也只是为了进行系统测试。但事实证明,这种想法是错误的!

Windows 10操作系统的公开发行版是由微软公司免费提供给广大用户的,而公司还是想办法在该系统中加入了键盘记录器。微软公司在今年的七月份正式发布了Windows 10操作系统,Windows 10免费升级这样前所未有的尝试对Windows 10装机量的急剧上升功不可没。在Windows 10发布仅仅48小时后,其全球装机量就已经超过了1400万台,刷新了Windows多项操作系统更新的纪录。截至上周五,Windows 10的装机量已经突破8100万台。这也导致很多人相信Windows 10实际的装机量早已超过这个数据。,但是免费的系统升级对于用户而言并不意味着真正的“免费”。

是的,除了一大堆的隐私问题以外,当你使用键盘,语音,屏幕,鼠标以及写字板的时候,在该系统中还会有一个软件组件来跟踪你的操作行为。但是,具体的情况可能会比你想象的更加复杂。

Windows 10的键盘记录器不仅仅只是一个简单的“键盘记录器”。

实际上,这个系统组件其实并不是一个键盘记录器,从某种程度上来说,这是一款恶意软件。

而且情况可能比我们想象的更为严重,微软公司曾公开表示:“当你通过语音,手写板,或键盘来与你的Windows设备进行交互,微软公司会收集你的语音信息,手写记录,以及键盘的输入数据,除此之外,我们还会收集你的日历信息以及通讯录信息等等。”

你可能会非常担心个人的隐私问题,但是请不用紧张,因为好消息就是你可以关掉这个键盘记录功能。

下面,我们将告诉大家如何关闭这个键盘记录器

-点击“开始菜单”(Start Menu),然后选择“设置“(Settings)。

-点击“隐私设置“(Privacy),你将会在菜单的最后一行找到这个设置选项。

-当你进入了“隐私设置“(Privacy)的菜单之后,点击“普通设置”(General)。

-找到“将我的输入数据发送给微软,以帮助开发人员提升用户输入和手写的体验度。”(Send Microsoft info about how I write to help use improve typing and writing in the future),然后点击按钮将该功能关闭。

-现在,请进入左侧的“语音,笔墨,和输入”( Speech, Inking and Typing)菜单,然后点击“停止收集有关我的信息”( Stop getting to know me)。这将会帮助你阻止Cortana语音助手对你语音信息的跟踪。

由于Windows 10操作系统中存在一系列的隐私问题,微软公司也因此名声扫地。

在大约一周之前,微软公司还将Windows 10操作系统中的数据采集功能和用户行为跟踪功能添加进了Window 7和Windows 8操作系统中,这样的行为也使得很多用户开始担心他们的隐私信息了。

但是,你同样可以阻止Window 7和Window 8操作系统对你的“监视行为”。点击这里获取详情。
