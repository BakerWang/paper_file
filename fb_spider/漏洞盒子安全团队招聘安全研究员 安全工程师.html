漏洞盒子安全团队招聘安全研究员、安全工程师

漏洞盒子 2014-11-25 共101184人围观 ，发现 28 个不明物体 安全招聘

企业级安全服务产品 – 漏洞盒子，致力于为客户提供完备的下一代互联网安全服务。业务上线以来目前已与多家大型知名企业合作，由于业务发展迅速，现诚招安全岗位同学，待遇优厚，优秀人才将纳入期权计划。

‍

安全研究员（Web||移动方向）（接受应届及实习）‍

工作描述：

1. 跟踪最新的全球安全动态并进行漏洞研究，包含Web，服务器，移动端等各个层面；
2. 熟悉Python编程，能够编写程序辅助进行研究或分析；
3. 成为漏洞盒子安全研究团队的一员，将研究成果转化为公司可使用的产品或服务。

职位要求：

1. 在Web安全，服务端安全，移动安全至少其中一方面有较深入的了解和实战经验；
2. 经常关注国外最新安全动态，能够流畅的阅读国外资讯或技术Paper；
2. 具备优秀的业务思维、团队思维、极客思维；
3. 熟练Python，能用Python完成基本的开发任务；
4. 擅长Linux操作；

加分项：

1. 积极参与国内外漏洞奖励计划并提交过有效漏洞；
2. 有自己的技术Blog，Github，参与或独立完成过开源项目；

高级安全工程师 （接受应届及实习）

工作职责：

1、负责对客户网络、系统进行安全评估和安全加固
2. 在出现网络攻击或安全事件时，提供紧急响应服务，帮助用户恢复系统及调查取证；
3. 针对客户网络架构，建议合理的网络安全解决方案；
4. 能够解决客户日常安全问题。

职位要求：

1. 具有1年以上工作经验，1年以上安全服务相关经验，2个以上的相关项目经验；
2. 熟悉多种安全技术，掌握或熟悉各种攻击与防护技术；
3. 表达能力佳，能够独立完成对用户的现场培训；
4. 具备较强的动手能力；
5. 敬业、正直、诚实；
6. 喜欢学习新知识、乐于接受新事物。

加分项：

1. 熟悉多种安全技术，掌握或熟悉各种攻击与防护技术；
2. 具有一定的编程能力，可以自己编写简单的攻击和检测程序；
3. 有安全服务项目经验；
4. 熟悉安全标准；
5. 有安全、网络、系统、数据库国际认证证书者，如CCNP、CCIE、CISP、CISSP等认证者优先考虑。

简历请发送至：hr@freebuf.com

‍‍‍‍工作地点：上海浦东国家级软件园‍‍‍‍

哦，对了。以上岗位：学历不限
