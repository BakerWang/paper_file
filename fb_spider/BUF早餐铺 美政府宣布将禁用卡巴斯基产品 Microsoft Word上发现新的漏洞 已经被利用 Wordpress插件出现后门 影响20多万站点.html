今天是 9 月 15 日星期五，今天的安全早报主要内容有：美国政府宣布将禁止使用卡巴斯基的产品；Microsoft Word 上发现新的漏洞，并已经被利用；Wordpress 插件中出现后门，影响20多万安装 WordPress 的站点；漏洞收购企业Zerodium表示向Tor浏览器项目提供100万美元0day漏洞奖励；欧盟委员会发布新的网络安全政策建议，指定“欧洲网络安全机构”处理网络威胁和攻击。


【安全漏洞】

Microsoft Word 上发现新的漏洞，并已经被利用

微软在周二的例行更新中，修补了 Microsoft Word中 的一个重要漏洞。并且据安全公司发现，这个新的 0 day 漏洞（CVE-2017-8759）已经被用来进行有针对性的攻击——安装一个名为 FinFisher 的恶意软件。

微软最近一直处于很忙碌的状态，恶意软件的开发者使用各种社工的手段诱骗用户。事实上，虽然通过驱动下载的活动降低了很多，但是恶意邮件一直是最主要的威胁。在这里也建议windows用户进行安全更新，并打上最新的安全补丁。

每当一个新的 0day 漏洞暴露出来，网络犯罪分子都会急于将其打造为自己的攻击武器，这意味着规模虽小但有针对性的攻击事件都会变得有广泛性，波及面也会更大。

[malwarebytes]

WordPress 插件中出现后门，影响20多万安装 WordPress 的站点

在过去的两个半月里，一个名为 Display Widgets 的 WordPress 插件，在WordPress 站点上留了一个后门。在 Display Widgets 2.6.1版本（6月30日发布）和2.63版本（9月2日）上发现了它的后门代码。WordPress.org 团队赶忙进行查看并从插件存储库中删除了该插件。当其被删除的时候，已经有20000个站点安装了该插件，但是无法确定有多少的更新包中包含恶意代码。

更令人惊讶的是，WordPress 的工作人员已经在此之前连续三次因为该插件违规而删除该插件。昨天，研究人员已经确认了这个恶意插件主人的身份。

【国际实事】

美国政府宣布将禁止使用卡巴斯基的产品

近日，美国国土安全部表示，禁止在政府电脑中使用卡巴斯基实验室的安全软件。参议院情报委员会表示卡巴斯基的杀毒软件可能与俄罗斯政府有秘密关系。根据美国国土安全部的指示，政府机构要在未来30天内识别出哪些设备上有卡巴斯基产品，并制定相关详细计划，在接下来的60天内删除使用中的卡巴斯基产品，并在接下来的90天内完全停止使用卡巴斯基的安全产品。

美国国土安全部同时表示，卡巴斯基可以对该判决表示上述，尽管专家认为这是徒劳的，自从美国政府对卡巴斯基的封杀传闻浮出水面之后，卡巴斯基实验室一直在强烈否认对此的任何指责。据悉，一个月前联邦调查局一直在与美国私密部门会面，要求美国公司抵制卡巴斯基产品。上周，美国公司百思买也停止使用了卡巴斯基的安全产品。

[bleepingcomputer]

欧盟委员会发布新的网络安全政策建议，指定“欧洲网络安全机构”处理网络威胁和攻击

近日，欧盟委员会主席 Euro-Claude Juncker 在欧盟的演讲中提出了这个建议，提出要建立“欧洲网络安全机构”。Juncker 指出，欧洲正在面临一个很危险的境地——每天都要有4000次 ransomware攻击。欧洲需要对此做出防御。需要建立起“欧洲网络安全机构”，帮助欧洲防范这种攻击，网络攻击可以危害到民主和政治的稳定，所以它要比枪支和坦克更为危险。

然而在讲话中，它并没有提到ENISA（欧盟网络和安全机构），这是一个位于希腊的欧盟机构，成立于2004年。但后来的一个委员会官员表示，Juncker 是想扩大它的作用，可以协调二十八个国家集团的网络活动。近几年，欧盟已经进行了几项与网络安全相关的实验性活动，战略中的一些计划还表示要加强网络安全的培训，加强执法的强度，加快网络威胁信息共享，这些都是必不可少的步骤。

[infosecurity-magazine]

CBI向英国政府警告，及时保护英国的数字经济，防止进入“数据悬崖边缘”

CBI 今天向政府警告，如若不加快努力加密 post-Brexit 数据流来保护英国的”数字经济“，英国将会进入“数据悬崖的边缘”。尽管政府通过了“数据保护法案”，将 GDPR 引入了英国国家法律中，但在英国2019年3月29日正式离开欧盟之后，确保数据稳定就显得很重要。保护数字经济也确实很重要，毕竟它价值2400亿英镑。这不仅仅是一个学术论证，而且还会实实在在影响到英国的工作，成长和繁荣。

然而专家对此又表示出了焦虑，因为如果给予国家这么大的监视权力，欧盟公民的数据可能会被英国当局监视。

[infosecurity-magazine]

【行业动态】

漏洞收购企业Zerodium表示向Tor浏览器项目提供100万美元0day漏洞奖励项目

看起来对于 Tor浏览器的漏洞攻击一直处于高需求状态，所以也就有人为此准备了100万美元。 Zerodium是一家专门从事漏洞收购业务的公司，近日宣布，将为linux 和 windows 操作系统上最受欢迎的 Tor 浏览器支付 100万美元的漏洞奖励金。Tor 浏览器用户最早知晓了这个消息，尤其是那些通过 Tor 浏览器保护他们隐私的用户。

0 day漏洞收购平台也在其网站上发布了详细的规则和支付细节，那些不需要 javascript的 Tor 漏洞将要比那些需要用到 javascript 的漏洞高两倍。该公司还清楚的提到，必须要使用远程执行代码来进行漏洞攻击，最初的攻击目标应该是一个网页，并且要支持最新版本的 Tor 浏览器。而且，0day攻击不能用到除了访问网站之外的任何用户交互方式。其他的攻击媒介也不符合规定，比如恶意文件的传送，ZERODIUM可以自行做出收购的判定。
