百度纵深防御团队简介

下一代web纵深防御系统是百度安全旗下一款面向中大型电商、互联网金融、游戏、教育等互联网企业的web安全整体解决方案，突破传统安全基于边界防护的设计理念，从网络、主机、数据库层面，依托百度人工智能技术以及虚拟机执行技术，结合百度安全大数据、威胁情报提供全方位的web纵深防护，保护企业核心web业务不被黑产网络攻击中断，保障企业核心业务数据不被黑产窃取。帮助企业建立完整的web防护体系，从传统的边界防护过渡到新一代的基于预测、检测、协同、防御、响应、溯源理念的web纵深防御

产品主页

百度纵深防御 http://xi.baidu.com/

WEBDIR+      http://scanner.baidu.com/

百度纵深防御团队现诚招以下岗位：

安全研发工程师_北京

工作职责:

- 负责百度纵深防御相关产品的研发工作

职位要求：

- 熟练掌握 Linux 平台的各种开发技能

- 精通一种或几种以下语言，PHP / Python / Java 等等 

- 熟悉常用算法和数据结构，熟悉网络编程、多线程编程技术 

- 具有数据库设计和优化能力，精通MySQL数据库 

- 善于学习和运用新知识，具有良好的分析和解决问题能力  

- 具有良好的团队合作精神和积极主动的沟通意识

- 有安全产品开发背景优先

- 本科或以上学历，计算机相关专业

实习安全研发工程师

工作职责:

- 负责百度纵深防御安全相关产品的研发工作

职责要求:

- 计算机科学及相关专业本科或以上学历

- 熟悉常用的数据结构和算法，熟悉网络编程和多线程编技术

- 熟悉Python、PHP、C和C++至少一种语言，熟悉常用脚本语言

- 理解Linux操作系统，熟悉TCP/IP等常见网络协议

- 有较好的团队协作及沟通能力

- 每周实习四天以上，至少实习三个月
