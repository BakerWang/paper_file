第二代树莓派全新发布：900MHz四核ARM Cortex-A7，1G内存，支持Win10

anymous 2015-02-03 +5 共368515人围观 ，发现 34 个不明物体 极客资讯

树莓派号称是世界上最便宜的电脑，没有之一，在之前着实火了一段时间。现在，它迎来了第二代产品——新树莓派2B型发布，900MHz四核ARM Cortex-A7，1G内存，支持Windows 10。

全新特性

官网链接

极客没有,DIY工具有可以升级了，新版本价格仍然不变，35$。（到了中国，会便宜的）

树莓PI2B型是第二代Raspberry Pi。它取代了原来的在2014二月的RaspberryPi 1B +型。

相比于Raspberry Pi 1的不同有

900MHz四核ARM Cortex-A7 CPU
1GB的RAM

与（PI1）B+型相比它也有：

4个USB端口
40个GPIO引脚
完整的HDMI端口
以太网端口
3.5mm音频 jack和复合视频  //他俩是一个各接口，不太了解此接口，姑且这么写了。
相机接口（CSI）
显示接口（DSI）
微型SD卡插槽
videocoreIV 3D图形核心

因为它有一个ARMv7处理器，它可以运行全范围的ARM GNU /Linux分支，包括Snappy Ubuntu Core，以及微软Windows 10（更多信息请看官方博客）。

Raspberry Pi2具有相同的因素与（PI1）模型B +相比和完全兼容Raspberry Pi1。

校园款

我们推荐的树莓PI 2模型B在学校中的应用：它提供了更多的灵活性相比（PI 1）A+型，对嵌入式项目和需要非常低的功率的项目更有用。

说些官方博客的简要，就不每句都翻译了

接口位置和功能未变，仍然使用5v的micro usb电源
由于更换处理器，需要刷入新的镜像
Raspberry Pi 2的win10 将免费提供给制造商
官方团队非营利性组织，自从推出原始B型，花了大量的金钱和精力去优化软件，解决硬件兼容性问题
向官方团队致敬，为了计算机教育着实做了巨大贡献！

常见问题解答（官网陆续更新中）

会停止Raspberry Pi 1 Model B and B+吗？

因为有很多工业客户暂时坚持，所以仍会继续，仍然35$

Raspberry Pi A+型呢？

在2015年底前还没有计划

还在用videocore？

是的。videocore IV 3D是唯一的公开记录的基于ARM的SoC的3D图形核心，我们想让Raspberry Pi在更开放
