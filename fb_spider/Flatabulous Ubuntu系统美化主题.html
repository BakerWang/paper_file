Flatabulous：Ubuntu系统美化主题

0xroot 2015-10-27 共417844人围观 ，发现 17 个不明物体 工具

Flatabulous是针对Ubuntu和其他基于Linux、debian的Flat主题。

Flatabulous基于Ultra-Flat主题。特别感谢@steftrikia和Satyajit Sahoo提供的原始版本。

安装

这个主题特别为Ubuntu个体环境设计，因为目前我只在Ubuntu上进行了测试。它可能无法在其他Linux环境中正常工作。

终端显示

安装主题的第一步是安装Ubuntu tweak tool，安装命令如下： 

sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak

或者进入网站，下载.deb文件。

接下来你可以点击这里下载主题。如果你是开发或者设计师，欢迎对它进行修改完善。

在home目录中创建一个名为.theme的文件夹。这样Flatabulous就可以通过开启终端由一种非常简单的方式被创建，输入

mkdir .theme

这是一个隐藏目录，想要在Nautilus中查看这个目录，可以在键盘中按下 `Ctrl/⌘ + H`。

你也可以选择另一种方式，即在/usr/share/themes/文件夹中放置extracted文件。

Flat图标

Dashboard

我使用了ultra-flat主题的图标。它可以使用推荐的蓝色、或者橙色、或者绿色。可以使用如下命令进行安装： 

sudo add-apt-repository ppa:noobslab/icons
sudo apt-get update
sudo apt-get install ultra-flat-ico

当然，也可以根据自己的喜好使用其他命令进行安装： 

sudo apt-get install ultra-flat-icons-orange

或者： 

sudo apt-get install ultra-flat-icons-green

我推荐使用这些flat图标，你也可以使用Numix和Flattr图标。在搜索里找到Ubuntu Tweak并启动它。tweaks的标签下有主题选项，在主题中选择Flatabulous主题，在图标设置中选择flat-icons。重启你的电脑，然后你就可以体验Flatabulous啦！
