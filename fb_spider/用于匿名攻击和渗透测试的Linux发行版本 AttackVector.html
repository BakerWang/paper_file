用于匿名攻击和渗透测试的Linux发行版本-AttackVector

LinuxSpace 2013-05-16 共192861人围观 ，发现 17 个不明物体 工具

AttackVector Linux是一个专注于安全、匿名、渗透的Linux发行版本。它是基于kaliLinux+TAILSLinux，当然他们都属于Debian流。

设计理念：阴（TAILS） 阳（Kali）

TAILS是从下往上进行加密的，是一个可使用户匿名的设计，这样一来nmap可以试图通过Tor来实现，而无需UDP。使AttackVector Linux拥有了匿名攻击的能力。

当采取攻击时，同时会警告用户，来方便不愿透露用户信息的用户。

Kali 则是通过修改内核，加入了许多网络设备驱动程序等等。

这两个项目是不同的设计理念，可能会存在避免不了的冲突。尽管如此，AttackVector Linux的目的是，将它们互补性地整合成一个OS。

特点
匿名性    TAILS

隐私性    SRWare Iron

密码恢复 hashkill

加密技术 DaKaRand

渗透工具 Kali

架构说明

1.安装kali 

apt-get install git live-build cdebootstrap kali-archive-keyring
cd /tmp
git clone git://git.kali.org/live-build-config.git
apt-get remove libdebian-installer4
apt-get install libdebian-installer4

2.复制git代码库[ distro / kali / config ] 

git clone git://github.com/ksoona/attackvector.git
cp attackvector/distro/kali/config /tmp/live-build-config/config
cd /tmp/live-build-config

3.构建LIVE 

lb clean --purge
dpkg --add-architecture amd64
lb config --architecture amd64 --mirror-binary http://http.kali.org/kali --mirror-binary-security http://security.kali.org/kali-security --apt-options "--force-yes --yes"
lb build
