BackBox Linux 4.0 初体验使用小记

帅气凌云 2014-10-23 +5 共344051人围观 ，发现 29 个不明物体 工具

BackBox是基于Ubuntu的Linux发行版，它是一款用于网络渗透测试及安全评估的操作系统。新版本4.0的更新包括Linux 3.13内核，EFI 模式，匿名模式，LVM +磁盘加密程序，隐私的补充和armhf Debian软件包。本文将与大家分享BackBox Linux 4.0的使用初体验。

1.中文输入法缺失？

以往backbox linux各版本都比较人性化，默认安装有ibus和中文拼音非常方便。

安装IBus框架

sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4

IBus拼音

sudo apt-get install ibus-pinyin

2.VPN代理功能缺失？

与往常不同的是新版本精简不少功能和程序。国内Tor翻+墙不容易（VPN功能也缺失拉，学习新技能受阻）

根据需求安装（常用的pptp）

apt-get install network-manager-openvpn-gnome
apt-get install network-manager-pptp
apt-get install network-manager-pptp-gnome
apt-get install network-manager-strongswan
apt-get install network-manager-vpnc
apt-get install network-manager-vpnc-gnome

3.菜单功能变化？

感觉视觉方便是有所提供，不过实用性不足以往。展开多级菜单要点到小三角才~

4.工具软件源和安装目录有很大变化？

新软件源：cat /etc/apt/sources.list.d/backbox-four-trusty.list
deb http://ppa.launchpad.net/backbox/four/ubuntu trusty main
# deb-src http://ppa.launchpad.net/backbox/four/ubuntu trusty main
旧的软件源: /etc/apt/sources.list

新安装路径：/opt/

旧安装路径：/opt/backbox/

5.sqlmap输出目录改变？

新的默认输出目录在当前用根目录下的.sqlmap(隐藏目录下）

6.中文输入法设置？

通过第一步我们已经安装拉ibus框架和ibus-pinyin输入法。下面我们开始设置～

菜单->所有设置(右下角，开关图标）->语言设置（选择系统键盘输入ibus）->其他选项（键盘输入-输入类型-显示所有选择中文）->添加中文拼音->重启系统->OK!
