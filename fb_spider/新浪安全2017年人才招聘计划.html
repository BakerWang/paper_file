新浪安全2017年人才招聘计划

-吗_啡- 2017-04-19 共229453人围观 ，发现 23 个不明物体 安全招聘

新浪公司是一家服务于中国及全球华人社群的网络媒体公司。新浪通过门户网站新浪网(SINA.com)、移动门户手机新浪网(SINA.cn)和社交网络服务及微博客服务新浪微博(Weibo.com)组成的数字媒体网络，帮助广大用户通过互联网和移动设备获得专业媒体和用户自生成的多媒体内容(UGC)并与友人进行兴趣分享。  

工作地点：北京市海淀区西北旺东路10号院西区8号楼新浪总部大厦

招聘职位：

安全运维工程师

薪资 8K-15K/月

岗位职责：

1. 参与公司安全相关系统搭建及管理

2. 参与公司服务器安全防护体系的设计及实施

3. 协助安全事件的调查，及时响应并处理突发安全事件

4. 跟踪和分析新的安全漏洞和安全技术

5. 负责公司相关服务器及安全设备的稳定运行与日常维护任务

职位要求：

1. 熟悉主流操作系统、网络设备、数据库的安全配置及加固;

2. 熟悉黑客防御的技术手段，有丰富的安全事件应急处理经验;

3. 熟悉防火墙、防病毒、IDS、日志审计、堡垒主机，防泄漏等安全产品的部署及使用;

4. 熟悉信息安全技术与产品、积极跟进国内外信息安全发展趋势；

5. 能恪守信息安全职业道德

加分项：

1. 具有良好的学习能力,有很强责任心、耐心和沟通能力优先;

2. 有安全事件挖掘、安全事件分析经验优先；

3. 能熟练操作shell、python等脚本语言

4. 3年以上的工作经验，有大型互联网从业经验优先考虑。

移动端安全工程师

薪资 15K-25K/月

岗位职责：

1、负责公司移动APP的安全，挖掘APP漏洞、并提供安全解决方案，协助开发人员修复安全问题； 

2、负责漏洞扫描平台的搭建与维护，定期对公司APP应用进行安全扫描与渗透测试；

3、负责APP安全技术研究，参与完善流程、安全架构方案、技术标准；

4、负责对公司开发、测试人员进行安全开发培训，提高安全编码水平；

任职资格：

1. 3年以上工作经验，2年以上APP相关安全测试经验与逆向分析经验（Android或IOS）；熟悉APP客户端安全架构、APP系统全生命周期安全管理，以及熟悉移动应用部署架构；

2. 精通app客户端的反编译、防动态调试、透明加密、防劫持、内存保护、进程监控等技术；

3. 精通iOS/Android 客户端与服务器通讯机制及加解密、算法等安全防护技术；

4. 掌握java/C等语言，能够熟练进行Android或Linux程序编写；具备独立编写脚本的能力。

以上岗位工作地点为北京新浪总部大厦

有意向者请发送简历到：security@staff.sina.com.cn（请注明来自FreeBuf和应聘岗位）
