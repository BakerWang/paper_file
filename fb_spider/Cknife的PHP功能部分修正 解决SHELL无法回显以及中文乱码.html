这只是第一部分,回显中有中文依旧会乱码，于是就有了后半部分文章

中文乱码解决

抓包,得到数据,可以看到，本地模拟的情况下。system函数执行的回显是乱码的。

后来经过测试,是编码的问题 ,把编码修改成utf-8即可显示中文啦~

下面是具体操作

并且需要把system函数修改成exec函数，用来把回显赋值给变量

修改部分

exec($r." 2>&1",$retn,$retn2);
foreach($retn as $res) {
    $res = iconv("GBK", "UTF-8", $res);
    echo $res;
    echo "\n";
};

即可正常显示中文
