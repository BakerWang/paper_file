Lalin：一款为Kali集成各种安全工具的懒人工具包

hey你的拔粪宝 2017-08-03 +10 共301936人围观 ，发现 18 个不明物体 工具资讯

Lalin是一个Kali专用的懒人工具包，集成了众多较新的工具以及一些基础功能，目前与Kali rolling兼容，适合新手及非深度Kali用户使用。


安装及使用

git clone https://github.com/Screetsec/LALIN.git
cd LALIN/
sudo chmod +x Lalin.sh
sudo ./Lalin.sh

工具及功能介绍

该工具近一年未更新，故一些功能不推荐使用。

下面是一些使用截图，这里只介绍一些实（能）用的，其他功能还请自行安装体验。

1.Update Kali

1和2已过时，且会修改sources.list，如果你的kali版本比较新，那么不要使用1和2！

3：更新kali，同apt-get update

4：更新并清理kali

5和6分别为还原和备份源，备份后/etc/apt中会生成一个sources.list.backup文件，以后恢复的时候可以直接使用该文件恢复。

2.Virtual Machine

如果你在虚拟机里运行Kali linux，这些可能会对你有帮助。输入序号即可安装对应工具（如选择2就会安装VMware-tools）

3.Browsers

如果不喜欢自带的浏览器，可以在这里输入序号安装Google Chrome及其他浏览器。

4.WIFu

这里包含了一些WIFI破解相关的工具，有些没有被Kali集成，按需安装！

5.Exploit-db

1：更新Exploitdb

2：搜索，选择2会提示输入关键词，搜索完成会弹出新窗口显示结果

6.sniffing&MITM

这里包含了流量嗅探和中间人攻击的工具，按需安装。

7.Firefox security toolkit

这会在firefox上添加拓展，让你的firefox成为渗透利器！详情点击这里。

14.Fixing all problem
