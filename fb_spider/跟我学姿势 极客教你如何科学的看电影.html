跟我学姿势：极客教你如何科学的看电影

mobier 2014-08-01 +15 共398100人围观 ，发现 50 个不明物体 头条极客

‍‍‍‍‍‍‍‍

‍‍俗话说自己动手丰衣足食，小米盒子之类的弱爆了，极客看片自然有极客的做法‍‍！相信看完本篇狠涨姿势的文章，你就可以像我一样吃着火锅唱着歌，喝着饮料吃着爆米花，用树莓派播放着nas下载好自己喜欢看的（cang）电（jing）影（kong）了！‍‍‍

nas部分
‍‍

1、U盘容量大于64m即可
2、显示器（或者之前就吧biso usb启动设置为第一项）
3、闲置的电脑、能正常开机即可
4、相关文件

废话真多，那就开始吧打开win32diskimager、将xx.img烧录到U盘

xx.img:http://pan.baidu.com/s/1gdGIff1

烧录软件：http://psoft.xpgod.com:801/small/win32diskimager_xpgod.rar

‍‍进主板的bios设置从U盘启动,装过系统应该都会,因为每个主板的设置方法不同,不会就百度。‍‍以下是进入到系统后.最开始有两个选择点。可以不做选择（因为过一段时间默认选择缺省设置）‍‍‍‍


‍‍然后等一两分钟、用群晖助手扫下网络查找会到1台nas的提示、右键安装
群晖助手http://global.download.synology. … tSetup-5.0-4448.exe
pat文件选用4482以上的版本。根据硬件选择:http://global.download.synology.com/download/DSM/5.0/
安装完之后、联机、用户密码刚刚设置的、如果没有设置用户名:admin密码:空‍‍

.

‍‍华丽丽的DSM5.0登陆界面出现了‍‍

‍

‍‍剩下的就是折腾了
由于群晖DOWNLOAD STATION 下载速度慢不太(la)好(ji)用,还是用迅雷的靠谱
通过nas网页文件管理吧迅雷固件解压上传上去
固件地址:http://g.xunlei.com/thread-128-1-1.html
‍‍打开控制面板-终端机-打开 telnet  or  ssh‍
登陆用户名:root密码:你自己设置 然后执行./volume1/刚刚上传地方/portal
注意启动时输出有一行：‍‍‍‍

‍

THE ACTIVE CODE IS: 42SND1

‍‍

‍‍“42SND1”为迅雷离线激活码‍‍
也可以在浏览器中输入http://NAS地址:9000/getsysinfo（IP替换为你路由器的IP，端口换为你的控制端口），会看到类似如下信息：
[ 0, 1, 1, 0, "42SND1",1, "201_2.1.3.121", "", 1 ]
其中有用的几项为：‍‍

‍‍

第一项：0表示返回结果成功
第二项：1表示检测网络正常，0表示检测网络异常
第四项：1表示已绑定成功，0表示未绑定
第五项：未绑定的情况下，为绑定的需要的激活码
第六项：1表示磁盘挂载检测成功，0表示磁盘挂载检测失败

‍‍之后进入迅雷远程下载页面并登录‍‍‍‍http://yuancheng.xunlei.com，添加NAS设备并填入刚才查到的激活码绑定NAS。

‍‍绑定完成之后就可以下载喜欢的(cang)电(jing)影(kong)了！‍‍‍‍‍‍‍‍


‍‍以下部分可以咯过!‍‍‍‍

这部分的目的是吧nas直接放进弱电箱、脱了机箱困扰、但具有一定的风险性！

吧机箱连接主板的线拔了、记着开机（power SW标签）连接两个引脚的位置、开机拿钥匙或者其他金属物体连接两个引脚即可！‍‍‍‍‍‍

‍‍

‍

‍‍将主板、电源、硬盘上的螺丝拧开、拿出来。‍‍‍‍‍

‍


‍

‍‍由于是老电脑了、一打开全是灰、建议是拿无水酒精、牙刷进行清理！‍‍‍

‍


‍

‍‍然后放进弱电箱、打孔用螺丝吧主板、硬盘固定住！然后就让他在弱电箱里吃灰！‍‍‍

‍


‍‍接下来是树莓派部分‍‍

1、树莓派
2、hdmi链接电视（主要是看电视接口、大多数都是hdmi）
3、网线or无线网卡
4、储存卡

‍‍‍‍通过‍‍‍‍Raspbmc的Installer烧到SD卡上、然后将烧录好的sd卡放入树莓派即可、
Raspbmc的Installer:http://212.187.212.70/bt/59a769cff6df1f5301def9b50ce4d54120ded179/data/raspbmc-win32.zip
‍‍如果局域网中没有DHCP服务器的话，还能事先设置好IP地址，需要注意的是，Raspbmc在树莓派上的安装全程需要网络支持，所以网络设置如果出错的话，则无法正常安装！需要手动设置ip。或者要连接无线网络请勾上Manually configure networking
‍‍Iacept tlicense agreem是同意协议意思、然后点击Install安装。‍‍‍‍

‍‍将sd卡放置在树莓派上开机即可、Raspbmc在树莓派上的安装时间取决于网络，因为SD卡中的只有安装用的引导程序，其他全部都需要从网络上进行下载、此过程无需操作，开树莓派连接上网络即可。

‍‍安装过程需要15-25分钟，如果你家里网络很慢的话，一晚上也说不定的哦……‍‍‍‍‍‍



‍‍安装好之后设置为中文、当然如果你看的懂英文可以跳过这一步‍‍

System—Settings–Appearance–Skin–Fonts–设置为Arial based
Appearance–International–Language–设置为Chinese(Simple）
Appearance–Region–设置为Beijing(12h)或者Beijing(24h）
Appearance–Character set–设置为Chinese Simplified(GBK)

‍

‍‍然后在视频-文件、添加视频源（NAS通过迅雷远程下载好的电影）、或者安装相关的插件在线直播体验机顶盒的感觉！‍‍‍

‍


‍‍如果你觉得做了以上的步骤还觉得不过瘾、或者说开电影没爆米花吃不过瘾、很好我们可以接着往下走！
作死部分需要准备‍‍

1、三个易拉罐
2、酒精or固态酒精(一般在药店就可以买到酒精和脱脂棉花)
3、美工刀
4、玉米、油or黄油

‍

把两个易拉罐用美工刀切至同一水平线‍‍‍

‍

把其中一个易拉罐做一些特殊处理、不然不好合上‍‍


‍‍用美工刀进行打孔！‍‍

‍‍酒精和棉花（可不放棉花）‍‍

‍‍大概的雏形‍‍

‍‍做爆米花的材料‍‍

‍‍然后将材料放入大一点易拉罐中并进行易拉罐开口处理‍‍

‍‍点燃酒精灯！坐等吃爆米花即可‍‍


‍‍然后边喝着饮料、边吃着爆米花、用树莓派播放着nas下载好自己喜欢看的（camg）电（jing）影（kong）、电视剧了！‍‍

‍‍‍‍‍‍‍‍
