国网智研院公开招聘信息安全渗透、逆向、监测分析人员

secsgri 2015-09-09 共233724人围观 ，发现 70 个不明物体 安全招聘

‍

‍

一、单位简介

国网智能电网研究院是国家电网公司直属科研单位，在直流输电、灵活交流输电、微电子、新材料、智能感知与量测、电网与环境控制等智能电网关键技术领域开展基础性、前瞻性研究。下设电力电子研究所、电工新材料及微电子研究所、直流输电技术研究所、信息通信研究所、计算及应用研究所等5个研究所，拥有美国研究院、欧洲研究院2个海外高端研发平。

在信息安全领域，国网智研院主要从事电网信息安全体系架构设计，关键核心安全防护装备和系统研发，开展智能电网信息系统安全渗透、漏洞挖掘、监测预警分析技术研究。同时负责国家电网公司“信息网络安全重点实验室”和“国家能源智能电网研发（实验）中心—信息安全技术分中心”的建设与运营。

二、招聘岗位、人数、工作地点

此次招聘共计3个岗位，需求人数6人。工作地点为‍北京‍（未来科技城国家电网园区）。

安全渗透高级工程师3人、逆向分析高级工程师2人、安全监测分析师1人。

三、应聘条件

1. 事业心和责任感强，求真务实，开拓进取，品行端正；

2. 有较强的学习能力和科研能力，勇于创新；

3. 本科及以上学历；

4. 3年及以上相关工作经历，年龄一般不超过35周岁，特别优秀可放宽到40岁。

安全渗透高级工程师

工作职责：

1. 负责信息系统安全渗透测试；

2. 负责安全攻防渗透相关技术研究；

3. 负责针对发现安全问题提出整改建议和解决方案；

4. 负责信息安全事件应急响应；

5. 负责跟踪国内外的安全事件，并进行严重性评估。

任职资格：

1. 本科及以上学历；计算机、信息安全及相关专业；

2. 熟悉攻击的各类技术及方法，了解各类操作系统、中间件、应用平台的弱点，对常见Web安全漏洞的原理、危害、利用方式及修复方案有较深的理解；

3. 熟悉渗透测试步骤、方法、流程，熟练使用常见的渗透测试工具，掌握安全漏洞发现和利用方法，具备一定的实战经验；

4. 熟练至少一种编程语言并可开发简单的安全工具；

5. 遵守法纪，无入侵破坏行为、无窃取数据行为、无参与黑色产业链经历；具有较强的责任感、具备能够独立的开展工作的能力，可独立完成单个项目的安全测试工作；

6. 有国内知名信息安全实验室工作经历者或国内网络攻防比赛取得名次者优先。

逆向分析高级工程师

工作职责：

1. 负责工控设备.智能设备漏洞挖掘；

2. 负责对恶意样本的逆向分析；

3. 负责信息安全逆向分析相关技术研究；

4. 参与病毒.木马相关网络安全事件的应急处置和调查分析；

5. 负责针对发现安全问题提出整改建议和解决方案。

任职资格：

1. 本科及以上学历；计算机、信息安全及相关专业；

2. 熟练掌握二进制代码逆向分析技术，熟悉x86和arm汇编语言；

3. 熟练掌握至少一门编程语言，能进行简单漏洞挖掘工具的编写；

4. 熟悉常见的漏洞原理以及利用，能够利用漏洞分析工具对漏洞的成因以及利用进行静态和动态的分析，对fuzz技术有比较深入的了解；

5. 有对恶意样本进行逆向分析的能力，了解linux操作系统基本命令的使用，能够在linux操作系统上搭建测试环境；

6. 熟悉电力常用通信协议、PLC开发和调试技术者优先。

安全监测分析师

工作职责：

1. 负责信息系统安全监测预警分析工作；

2. 负责信息安全监测预警分析相关技术研究，负责信息安全监测模型研究；

3. 参与公司信息安全事件分析及调查取证工作；

4. 负责跟踪国内外的安全动态和情报，编制分析报告。

任职资格：

1. 硕士及以上学历；计算机、信息安全、数学相关专业；

2. 熟悉信息安全攻击渗透、漏洞挖掘的技术原理和方法；

3. 熟悉多种网络安全技术，对各种主流的安全产品有较好的理论基础和实践经验，熟悉网络、操作系统、数据库、中间件等系统；

4. 3年以上信息安全分析或渗透工作经验，对信息安全异常情况敏感，能够独立创建安全事件场景，并设计分析模型；

5. 良好的英语阅读能力。

四、待遇

12K+，根据具体岗位和人员条件确定。

五、招聘程序

简历收集和资格审查、考试面试、确定录用人选。

六、联系方式

邮箱 sec#sgri.sgcc.com.cn（#请换成@，注明来自FreeBuf）
