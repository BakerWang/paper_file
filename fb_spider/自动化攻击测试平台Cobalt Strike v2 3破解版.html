自动化攻击测试平台Cobalt Strike v2.3破解版

Jumbo 2015-01-30 共185357人围观 ，发现 19 个不明物体 工具

Armitage是一款Java写的Metasploit图形界面的渗透测试软件，可以用它结合Metasploit已知的exploit来针对存在的漏洞进行自动化测试。渗透测试系统Bt5、kali LINX下集成免费版本Armitage，而Cobaltstrike则是它的商业版本。其图形界面非常友好，一键傻瓜化使用MSF高级功能，可以实现自动渗透测试。

小编：破解版仅用于尝鲜之用，请支持正版！

Cobalt Strike 2.3主要更新内容

+ 对攻击负载Beacon的多项特性改进
+ kerberos_ticket_use不需要输入参数
+ 新增Attacks -> Web Drive-by -> PowerShell Web Delivery. 
+ 多项bug修复
