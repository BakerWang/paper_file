ISIS开发新型加密安卓通讯软件

亲爱的兔子 2016-01-17 共149767人围观 ，发现 1 个不明物体 资讯

据报道：目前ISIS又开发了一款新的通信软件。

ISIS使用新款通信软件进行交流

这款应用程序被称为“Amaq代理应用程序”，同时该应用程序可以运行在安卓平台。那么这款应用到底是不是ISIS成员使用的加密通讯软件？他们具体使用软件里面的什么信息服务功能？这些可能都是西方情报机构目前想要弄清楚的问题，早前 “Ghost Security Group ”发现了一款安卓环境下的应用程序，而ISIS成员正是利用这款程序应用来进行交流。

Ghost Security Group组织主要由计算机专家组成。早前曾支持美国执法机构并在纽约和突尼斯阻止ISIS的恐怖活动。而对于ISIS最新开发的通讯加密应用程序，Ghost Security Group表示，该应用程序重新整合了其它同样的应用程序，最终被开发出来。目前在“匿名者”等黑客组织的攻击下，ISIS发现很难再使用Telegram或WhatsApp等短消息应用。不过恐怖分子依然可以使用Tumblr等网站沟通，从内部圈子中询问意见。

ISIS早前也曾使用通讯平台进行交流

早前在法国巴黎恐怖袭击事件中，当时比利时内政大臣Jan Jambon透露，袭击巴黎的恐怖组织成员常以PS4进行联络，原因是PS4难以被监察。通信专家也认为联网电子游戏机遥控联络比起WhatsApp更难追踪。 在上个月， Ghost Security Group以及安全情报专家就开始注意到社交媒体平台上一些用户开始共享一个应用程序，而这款应用程序就是ISIS开发的加密程序应用，该应用程序名称为“Amaq代理应用程序”。

这款应用程序主要被用来宣传和散播ISIS一些煽动宣传信息，而该应用主要通过Telegram以及Twitter平台发布，目前已经有很多下载链接已经作废。ISIS成员主要使用应用程序分享一些资料以及视频。虽然ISIS成员可以使用用这款应用进行沟通，它“可以进行加密通信，但和Telegram或其他应用相比较为原始。”

近日美国联邦调查局FBI局长James Comey在接受《60分钟》栏目采访时再次表示，他认为iPhone和Android手机的数据加密功能是凌驾于法律之上的，这种数据加密导致执法机构无法获取用户数据。即使拥有法院合法法院命令，智能手机上储存的数据也不能被获取，这也导致很多情况下调查工作无法展开。

早前希拉里曾建议，与其试图去破解加密信息，倒不如让美国政府发起一个类似于曼哈顿计划的项目，将政府和科技公司团结起来，这样执法部门才能阻止袭击的发生。
