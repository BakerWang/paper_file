完美世界信息安全部诚聘安全岗位

乐平 2016-04-20 共192284人围观 ，发现 13 个不明物体 安全招聘

2016年3月，完美世界私有化后和完美环球（影视）重组回归 A 股上市，使得在美上市八年的老牌游戏巨头正式回归A股。截至2015年9月底，完美世界共运营了包括《诛仙》、《射雕英雄传》、《最终幻想》等50多款各类游戏。而且拥有“星球迷航”、“无冬之夜”、“圣斗士星矢”的优质IP，以及3D游戏引擎技术。使得完美环球的影视、游戏平台也合二为一。

完美信息安全部团队负责整个完美集团所有安全业务，业务类型多样化所面临的威胁也是各式各样，工作富有挑战性。团队成员有安全测试、渗透、安全运维、开发、逆向等方向的小伙伴，部门有正在开发的安全平台，业务也处于快速发展阶段，安全很受公司重视，直线VP虚线 CEO 汇报关系，期待你的加入，一起成长。

WAF开发工程师

岗位职责：

1.        负责内部的WAF系统开发

职责要求：

1.        熟悉以下任意一种一种C 系语言开发：C 、C++

2.        熟悉Nginx架构和nginx-Lua，能对架构进行优化

3.        对 web 安全有一定的了解，了解web 攻防原理

优先条件：

1.        对技术有热情，学习能力强，有强烈的责任心、

2.        熟悉开源系统，有贡献开源系统代码、技术 blog

待遇：20 – 35K

Elasticsearch/spark 开发工程师

岗位职责：（ES、spark 可偏一样）

1.        负责安全 数据分析平台的开发工作

2.        负责ES、spark 集群的调优、运维工作

3.        负责数据 ETL 工作

职责要求：

1.        熟练ES 或 spark 相关技术，了解工作原理

2.        熟悉一下任意一种语言：python、java、scala

3.        能阅读英文文档

4.        对技术有热情，有较强的分析、动手解决技术问题能力

5.        对工作有热情，耐心、责任心强，具备沟通能力和团队意识

优先条件：任意

1.        有过海量数据处理、开发经验者优先

2.        有机器学习经验者优先

待遇：18- 35K

安全渗透工程师

岗位职责：

1.        负责上线安全测试、例行安全检查、渗透测试工作

2.        负责优化应急响应流程和突发事件的应急响应

3.        负责最新安全漏洞研究、跟进工作

职责要求：

1.        熟练掌握常见 web、系统安全漏洞原理、利用方式以及对解决方案有较深入理解

2.        精通 web 攻击方法，能独立研究发现新的漏洞

3.        熟悉脚本语言，可以将重复性的工作转换成自动或半自动化脚本实现

4.        对工作有热情，耐心、责任心强，具备沟通能力和团队意识

优先条件：任意

1.        有白盒审计能力和经验优先

2.        有后门分析能力和经验优先

3.        有 SDL 经验的优先

待遇：15 – 25k

安全运维工程师

岗位职责：

1.        负责公司的安全运维工作

2.        负责安全事件的响应、跟踪分析、挖掘工作

3.        负责内部安全系统、平台的运营工作

职责要求：

1.        熟悉 Linux、windows、网络设备运维管理

2.        熟悉应用服务、系统的深度安全加固

3.        熟悉恶意代码检测，清除

4.        熟悉常见漏洞攻击原理和入侵方式

5.        熟悉 python、shell 脚本语言，有脚本编写经验

6.        有工作热情，耐心、责任心强，沟通能力强

优先条件：任意

1.        有安全运维经验或应急响应经验优先

2.        善于探索，总结，优化流程优先

待遇：15 – 20k

安全实习生

岗位方向：

1.        安全运维方向：部门安全 运维工作、系统的规则更新工作

2.        渗透测试方向：参与部门安全测试、渗透测试工作

3.        开发方向：参与部门安全平台开发工作，前端、后端都需要

4.        数据分析方向：参与安全大数据系统开发、分析工作

职位要求：

1.        有上述方向的相关技能经验

2.        了解安全原理，与常见攻击方法

3.        对安全有热情，工作有责任心

4.        实习时间可以保持每周4天以上，最少3个月

优先条件：任意一项

1.        熟悉正则表达式编写，熟悉linux 系统，有脚本编写经验

2.        熟悉 python、java ，有开发经验

3.        熟悉 web 安全攻防，有渗透测试、安全测试经验

待遇：5K

福利待遇

1.        薪酬：15 – 35K（详见具体岗位薪酬），12月薪加2-6个月年底奖金

2.        每年定期调薪，优异者特殊调薪机制

3.        员工持股激励机制

4.        内部食堂，免费现磨咖啡、可乐等饮料

5.        每年国内、国外旅游，不定期的部门团建

6.        生日趴、相亲趴、各种游戏内测、福利大礼包

7.        无限制购买学习、研究资料、设备

8.        全球化公司，有外派出国（旧金山、阿姆斯特丹、马来西亚）等机会
