最新Mac OS X Yosemite：偷偷发送用户定位和浏览器搜索数据？

dawner 2014-10-24 +5 共116995人围观 ，发现 54 个不明物体 资讯

苹果最新的桌面操作系统名为Mac OS X 10.10 Yosemite，会在用户使用桌面搜索工具Spotlight时，默认在其不知晓的情况下发送用户搜索记录和定位数据到苹果升级服务器上，再次侵犯了用户的隐私。

周一，当用户意识到该公司关于Spotlight的隐私声明，即明确表明Mac OS X 10.10 Yosemite或是近期发布的IOS8系统，会把定位数据和搜索记录发回苹果的服务器处理时，该科技公司被公众强烈斥责。

苹果收集用户数据并转交微软

一方面，苹果决定顶住FBI的压力，强行默认启用加密硬盘。但另一方面，公司本身却将用户隐私置于危险之中（小编：原来童话里都是骗人的T_T）。苹果公司直率地承认了其服务内容，用户在Spotlight里搜索的数据也会被转到微软的Bing搜索引擎里。

苹果的“关于Spotlight隐私声明”文档：“当你使用Spotlight，Spotlight建议你进行的搜索查询选项和相关可用数据，会被发送给苹果”。“如果你设备上的定位服务开启，当你在Spotlight搜索时，它同样会将当时的相关地理数据发送给苹果。”

官方于周四发布Mac OS X 10.10 Yosemite，它允许Mac用户向他们自己的云盘上传和创建任何类型的文件，这意味着用户可以在iCloud盘存储所有类型文件，总大小不超过15GB。有了Yosemite，苹果开始统筹桌面和移动计算平台。

然而，这个现代科技巨头表示，Mac OS X 10.10 Yosemite的软件Spotlight，其所作所为都是为了提升Spotlight的引擎。但是，如果用户并不希望他们的数据被收集起来，他们可以关掉Spotlight建议和系统里的Bing搜索偏好。

如何关掉恶心的官方设置

据Fix-MacOSX.com报道，某名开发者还上传了一个Python脚本，可以防止苹果收集数据。你可以根据指令操作，按步骤关掉Spotlight搜索。

禁用“Spotlight建议”和“Bing搜索” 的过程如下:

System Preferences > Spotlight > Search Results

Safari也有一个名为“Spotlight建议”的设置，它与Spotlight的“Spotlight建议”是不同的，其使用了与Spotlight相同的机制，一旦被启用，Safari会将所有查询内容发送给苹果。

此前，你可能会认为自己已经禁用了“Spotlight建议，”但其实你还需要做如下工作来取消“Spotlight建议”：

Safari > Preferences > Search

来自苹果的回应

苹果已经发表声明，公司已经建立Spotlight搜索隐私保护机制，来抑制Spotlight发送给苹果的信息量。

苹果公司在iMore中声明：

我们正在切实地致力于保护用户的隐私，并且在我们的产品中实现隐私权。至于Spotlight建议，我们已经减少了其给苹果发送的信息量。另外，苹果不会保留用户设备的IP地址。Spotlight定位模糊，因此它并不会发送用户准确位置给苹果。Spotlight并没有给数据做上永久的标志，因此用户的搜索历史并不能被其他人（包括苹果）再现。苹果设备只会使用一个匿名临时session ID，持续15分钟以后该ID就被销毁。

我们也曾与微软密切合作，以此保护用户隐私。苹果只会把通用的搜索条件和城市级别的定位发送给Bing。微软并没有存储任何搜索内容，亦没有接收过用户的IP数据。最后，你也可以很容易地取消Spotlight建议，以及Spotlight里的Bing搜索或定位服务。

其实，Google和其他高科技公司也玩过相同的把戏。所以，用户只需要知道世上没有什么东西是免费的。Google、Facebook、新版的Mac OS X 10.10 Yosemite也许都不是免费的，因为你正在出售自己的个人信息。
