完美世界招聘各类安全人才

乐平 2015-09-22 共123993人围观 ，发现 24 个不明物体 安全招聘

完美世界（北京）网络技术有限公司（以下简称：完美世界）是中国领先的网络游戏开发商和运营商之一。公司成立于2004年，致力于以自主原创为根本，以研发优质的精品网游为宗旨，以推广东方文化于世界为长远目标。

完美世界现主要基于自主研发的Angelica 3D游戏引擎、Cube引擎以及Eparch 2D引擎为平台开发各类网络游戏，陆续推出了《完美世界》、《武林外传》、《完美世界国际版》、《诛仙》、《赤壁》、《热舞派对》、《口袋西游》、《神鬼传奇》、《梦幻诛仙》、《降龙之剑》、《神魔大陆》、《神鬼世界》、《倚天屠龙记》、《神雕侠侣》、《圣王》、《圣斗士星矢online》等网络游戏产品。

凭借强大的技术实力及富有创意的游戏设计能力，完美世界迅速推出广受大众欢迎的游戏产品，以满足全球用户不断变化的需求及市场发展。作为网络游戏产品提供商，完美世界正面向全球开展业务，全面拓展海外市场。目前，已经成功的将旗下部分产品出口到了海外一百多个国家和地区。

同时，完美世界在全球建立了自己的运营平台，在北美、欧洲和亚洲设有全资子公司自主运营游戏，并已成功将旗下游戏授权至亚洲、拉丁美洲、澳大利亚、新西兰、俄罗斯联邦及其它俄语地区等多个国家和地区的主要游戏运营商，为全球用户提供优质的互联网娱乐服务内容。

2007年7月26日，完美世界成功登陆美国纳斯达克股票市场（NASDAQ：PWRD），并于2009年1月2日登陆纳斯达克全球精选市场(NASDAQ Global Select Market)挂牌交易，成为中国最出色的互动娱乐企业之一。

But，以上都不那么重要，我想告诉你的是以下事实，希望你能考虑加入我们：

1.完美世界已经完成了美国私有化退市，接下来会在国内重新上市，钱景你懂的；
2.部门直线汇报 VP，虚线汇报 CEO，在公司的支持力度，和推动力你也懂的；
3.部门目前只有8个人，大量开拓性，平台化的工作正在推进，能力和职业发展你也懂的。

目前我们需要这些人才的加入，如果你想推荐或自荐下面没有的岗位，也请联系我，只要确实对公司安全发展有利，可以特设。引用团队某人的话：“这都不是事！”

安全运维工程师

岗位职责：

1. 负责公司安全系统运维工作

2. 负责安全事件跟踪分析、挖掘工作

3. 负责支持业务运维安全工作

职位要求：

1. 熟悉 Linux 、windows系统和运维

2. 熟悉 web 服务器、系统的安全加固

3. 熟悉恶意代码检测，清除

4. 熟悉常见漏洞攻击原理和入侵方式

5. 至少一年安全运维工作经验

加分项：

1. 有很强责任心、耐心和沟通能力优先

2. 有安全事件响应、跟踪经验优先

移动安全工程师

岗位职责：

1. 负责公司移动安全产品测试

2. 负责移动安全测试体系建立和维护

3. 负责移动安全测试技术研究、漏洞跟踪

4. 负责移动安全开发标准制定、培训工作

职位要求：

1. 熟悉APP的反编译、动态调试、加密、防劫持、安全加固等技术

2. 熟悉 APP架构、包括部署架构，了解对APP 生命周期安全管理

3. 熟悉IOS、Android 客户端和服务器端通信机制及防护技术

4. 至少熟悉 python、java、perl 等一门语言

5. 有一 年以上实际相关工作经验

加分项：

1. 有移动游戏安全研究经验优先

2. 有各平台漏洞提交、研究、分析报告优先

安全开发工程师

岗位职责：

1. 负责安全平台的功能开发

2. 负责安全平台的优化更新

职位要求：

1. 数量掌握 JAVA/Python中的一门语言，并熟悉常见WEB 开发框架

2. 熟悉常见安全漏洞原理，理解安全产品

3. 有扫描系统、WAF 、安全agent等系统开发经验优先

4. 熟悉 ELK、Spark 大数据生态系统的优先

5. 有二年以上实际开发相关工作经验

加分项：

1. 有安全产品开发经验优先

2. 有开源项目优先

安全数据分析工程师

岗位职责：

‍‍‍‍‍‍‍‍‍‍

1. 负责监控分析告警数据，快速响应和处置安全事件

‍‍‍‍

2. 负责业务数据挖掘，分析业务安全风险并推动处置‍‍‍‍

3. 负责安全数据挖掘，建立关联分析模型‍‍

职位要求：

1. 熟悉安全攻击，了解各类入侵方式

2. 熟悉黑产，了解黑产链条和技术手段

3. 熟悉相关分析工具和正则表达式编写能力

4. 熟悉 python/python/shell 某一类脚本编写

5. 有相关安全规则编写经验优先

6. 对安全有强烈的热情，善于思考有独立解决问题能力

加分项：

1. 有业务安全经验优先

2. 有规则分析、编写、建模经验优先

不合适我们的人：

1. 有黑产背景的人

2. 按部就班，养老混日子的人

3. 对安全工作没有激情的人

工作地点：北京（大屯路-完美世界大厦）

薪酬：15k-35k，14 -18薪

福利：

  1. 每年定期调薪，工作优异者有特殊调薪

  2. 生日趴、相亲趴、各种游戏内测、内部福利大礼包等等…

  3. 作为国内最成功的海外游戏出口商，我们承担海外公司安全支持责任，so 北美、日本等海外出差外派机会等你

  4. 公司内部食堂，提供早、中、晚餐加下午茶，免费的现磨咖啡、可乐这些就不用说了，标配

  5. 鼓励学习、研究，需要购买资源、书籍、设备，尽管提，全力支持

  6. 我们还有影视公司，要是安全工作烦了，也可以跨界演艺圈，不济也可以混混签名照（办公区常被用来影视取景，影视公司就在隔壁）‍‍‍‍
