青藤云安全招贤纳士啦

关于青藤云安全：

作为中国第一家专注于服务互联网企业客户的安全公司，青藤云安全致力于通过技术改革和商业模式上的创新，让安全对于企业而言不再是奢侈品。基于青藤自适应安全平台，任何互联网企业都可以在15分钟内构建出最符合自身需求的安全体系。

青藤自适应安全平台基于全球最先进的安全理念，通过自内而外的持续监控和与业务相结合的安全防护（而不是传统的基于边界的防火墙等产品），可自适应于私有云、混合云，公有云等不同基础架构，并提供给客户X光式的业务系统和云服务运作视图，使得安全人员甚至运维/ 开发都可以方便地了解安全情态并进行高效操作。 

我们的产品核心：自适应安全理念的诞生：http://news.163.com/15/0608/16/ARJODK1P00014AED.html

公司的创始人兼CEO张福，新一代安全理念——自适应安全的发起人，使企业安全运维人员可以清楚掌握企业内部系统异常运作和外部攻击行为，真正做到对多形态甚至高级攻击的快速相应恢复。他是国内最早的一批黑客之一，长期在知名企业如九城、盛大、 51.com、昆仑万维负责核心业务研发工作，在安全攻防领域有超过15年的实践经验，是国内屈指可数的打通互联网业务研发和安全体系两个领域的专家。青藤团队由安全专家和互联网技术高手组合而成，多数成员具备互联网一线业务研发和安全攻防的双重技术背景，平均年龄超过 30岁，多数具有至少两家知名企业工作经验。有来自于爱立信的分布式系统专家，有机器学习和人工智能方面的大牛，有”码龄“15 年的超级码农，还有国内最顶尖的黑客 /安全专家。

优秀的团队背景，先进的产品技术理念，以及上千亿美元的目标市场，使团队一成立就获得风投的青睐。公司从2014年8月创立以来，就获得来自真格基金，云天使基金，丰厚资本的联合天使投资 650万人民币。2015年8月我们获得来自宽带资本，红点投资的 1000 万美金的A轮融资，青藤云安全已与国内多家知名互联网企业达成合作。

我们是一家工程师文化驱动的创业团队，崇尚快乐生活，激情创业，希望团队每个伙伴都可以自我管理学习成长。

找个有前景有发展的工作，就来青藤，让我们一起推动互联网企业安全革命。

招聘岗位

安全顾问（北京、深圳）

岗位职责：

1. 负责售前顾问工作，配合销售人员参与投标项目，完成整个投标过程；

2. 与用户进行现场交流；

3. 为用户在项目启动前撰写和提供相关的技术文件，包括公司资料、技术建议书、长期规划建议等；

4. 负责和参与对用户的短期培训和讲座。

任职要求：

1. 大学本科及以上学历，1年以上的相关工作经验；

2. 具有规范编写专业文档的能力，包括申报材料、宣传软文、分析报告、解决方案、成功案例分析等；

3. 熟悉信息安全理论、主流信息安全技术、产品的应用部署及方案设计；   

4. 了解国家信息安全相关政策法规与标准规范。

优先条件：

有CISSP、CISA、ITIL  认证的优先。

薪资范围10k~20k

安全顾问（运营商方向）（北京）

岗位职责：

1. 负责售前顾问工作，配合销售人员参与投标项目，完成整个投标过程；

2. 与用户进行现场交流，包括初次交流、具体项目交流和投标交流； 

3.  为用户在项目启动前撰写和提供相关的技术文件，包括公司资料、技术建议书、长期规划建议等；

4. 参与投标项目，负责撰写投标方案技术文件部分和过程中的技术应答文件；

5.  负责和参与对用户的短期培训和讲座； 

6. 作为对销售部门售前过程的唯一接口，负责帮助销售协调测试和其他部门的资源；

7. 根据公司安排，支持各分支机构的重要项目。

任职要求：

1.   学历不限，1-3年的相关工作经验；

2.   安全背景；

3.   熟悉运营商业务场景和使用要求，了解运营商的采购过程。

薪资范围10k~20k

安全工程师（北京、武汉）

岗位职责：

1. 能够独立完成对用户的现场培训、应急响应、渗透测试、代码审计等工作；

2. 具有较强的独立动手能力和学习能力，能够独立完成产品项目的实施和交付； 

3.  有一定的脚本编写能力，能够将安全规则转化成自动化脚本实现。

任职要求：

1. 大学本科及以上学历；

2. 具有1年以上安全服务相关经验，5个以上的相关项目经验；   

3. 有Linux、Windows系统知识经验，能熟练使用  Linux 、Windows系统平台下各种应用系统，如：MSSQL、 Oracle 、Mysql、Apache、 Tomcat 、 Weblogic等；

4. 熟悉相关网络安全产品和安全服务，如防火墙、 IDS 、防病毒，漏洞评估工具，渗透测试，应急响应，系统安全评估，安全培训，代码审计服务等； 

5. 掌握一门编程语言，lua 、 python、 java等。

优先条件：

有CISSP、CISA、ITIL  认证的优先。

薪资范围10k~20k

产品经理（安全方向）（北京）

岗位职责：

1. 基于安全技术解决方案，结合用户需求和产品技术架构，完成产品/功能的概念设计和原型展示；   

2.  完成交付给开发和测试团队产品需求文档，功能定义，协同用户体验设计师完成视觉设计，交互设计等详细设计文档；

3. 设计运营后台，分析产品功能数据并持续跟进功能相关的运营工作。

任职要求：

1. 大学本科及以上学历，1-5年的相关工作经验；

2. 有网络安全相关技术基础；

3. 能够撰写需求文档和产品的原型，具有产品需求分析和规划能力；

4.  卓越的逻辑思维与分析能力，能够很好的规划和设计产品功能、使用流程与交互体验；

5. 有比较强的沟通协调能力，有很好的抗压能力。

优先条件：

1. 有过安全产品经验；

2. 熟练的英文读写能力。

薪资范围15k~25k

PHP开发工程师（北京）

岗位职责：

1. 支持青藤产品正常迭代的功能开发需求；

2. 支持CRM等后台系统的开发需求；

3. 对现有功能进行持续的优化和改进，不断提高产品的可维护性和系统性能。

任职要求：

1. 大学本科及以上学历；

2. 熟悉MySql数据库的设计和优化；

3. 精通PHP语言，掌握LNMP开发；

4. 熟悉MVC模式，熟练掌握Yaf框架者优先  ; 熟悉HTML/CSS/JS等前端技术，有bootstrap、  AngularJS 、Vue等前端框架使用经验者优先；

5. 熟悉OOP、可以独立承担程序模块的设计和开发工作；

6. 熟悉linux及其相关环境的配置和使用。

优先条件：

1. 有一线互联网公司工作经验或者成熟的线上项目开发经验的优先；           

2. 有热情有能力可以提供快速的成长空间。

薪资范围10k~20k

LinuxC++ （北京）

岗位职责：

1. 跟踪网络安全业界动态，从事恶意软件的研究；

2. 研究Linux 系统安全功能的技术难点和负责基础部分的开发；   

3. 负责安全终端产品的基础功能的维护和开发。

任职要求：

1. 大学本科及以上学历，1年以上的相关工作经验；

2. 计算机和网络安全、数学相关专业本科及以上学历，或者恶意软件检测相关项目3年及以上经验；

3.  熟悉Linux下系统程序编程和C/C++的开发，熟悉Python  ； 

4. 对数据结构有深入了解，理解OO思想（面向对象、设计模式、STL  使用等技术）； 

5. 良好的沟通、团队协作和管理能力。

优先条件：   

1. 具有软件调试及逆向分析能力；

2. 熟悉 Linux内核及开发。

薪资范围10k~20k

资深后台研发工程师（北京）

岗位职责：

开发和维护云安全核心系统后台及数据库。

任职要求：

1. 大学本科及以上学历，211/985优先考虑；

2. 熟悉Erlang/Java/C/C++语言之一，至少有一种扎实的编程能力；

3.  熟悉TCP/IP协议和网络编程；

4. 熟悉常见的分布式数据库,  如Riak/Hbase/Couchbase／MongoDB等等；

5.  熟悉Linux常用命令，调试工具，性能调优工具，有Linux平台开发经验；

6.  良好的编码风格，重视代码的质量和容错性，习惯单元测试。

优先条件:   

1.具有大数据量高负载产品/ 系统的设计或核心开发的经验；

2.拥有开源项目的开发经验；   

3.熟悉 Erlang 语言，熟悉 OTP  设计模式，有项目开发经验。

薪资范围15k~25k
