完美(世界)集团信息安全部诚聘(北京)

乐平 2016-02-21 共188337人围观 ，发现 29 个不明物体 安全招聘


2016年1月，完美环球以120亿元的交易对价收购了曾经的母公司完美世界100%股权，使得在美上市八年的老牌游戏巨头正式回归A股。截至2015年9月底，完美世界共运营了包括《诛仙》、《射雕英雄传》、《最终幻想》等50多款各类游戏。

完美信息安全部负责整个完美集团所有业务的安全保障工作，面临的业务类型、威胁多样化。安全团队内测试、运维、开发、逆向各种类型的人都有，扁平化、透明化管理，安全业务处于大力发展阶段，有很多初创以及具有挑战性的工作，期待你来加入，一起建设，一起成长。

高级安全开发工程师

岗位职责：

1.    负责开发公司的 主机安全防护平台

职责要求：

1.    熟练掌握Linux C/C++ 开发调试技能，有相关项目经验

2.    有过硬的代码能力，良好的编程习惯，规范的开发流程

3.    熟悉 linux 内核机制、网络编程、多核多线程 和常用算法与数据结构

4.    对工作有热情、责任心强，有团队意识

优先条件：

1.     有主机防护、监控类系统开发经验

2.     有高并发、大规模处理系统开发经验

3.     有安全产品开发背景优先

待遇：20 – 35K

大数据开发工程师

岗位职责：

1.    负责基于 ELasticsearch 的分析、告警系统研发

2.    负责调优 Elasticsearch 集群调优

3.    负责安全分析、建模

职责要求：

1.    熟练掌握JAVA、Python后端开发经验，如多线程网络等

2.    熟悉 ELasticsearch 、Lucenne 和相关生态组件（ELK、kafka、zookeeper）

3.     对工作有热情，责任心强，对数据分析、挖掘有兴趣，自我驱动型

优先条件：

1.     有搜索、监控、数据分析系统开发经验优先

2.     有数据挖掘、统计学知识优先

3.     有海量数据处理经验者优先

待遇：18- 35K

安全渗透工程师

岗位职责：

1.    负责上线安全测试、例行安全检查、渗透测试工作

2.    负责优化应急响应流程和突发事件的应急响应

3.    负责最新安全漏洞研究、跟进工作

职责要求：

1.    熟练掌握常见 web、系统安全漏洞原理、利用方式以及对解决方案有较深入理解

2.    精通 web 攻击方法，能独立研究发现新的漏洞

3.    熟悉脚本语言，可以将重复性的工作转换成自动或半自动化脚本实现

4.    对工作有热情，耐心、责任心强，具备沟通能力和团队意识

优先条件：

1.     有白盒审计能力和经验优先

2.     有后门分析能力和经验优先

待遇：15 – 25k

安全运维工程师

岗位职责：

1.    负责公司的安全运维工作

2.    负责安全事件的响应、跟踪分析、挖掘工作

3.    负责内部安全系统、平台的运营工作

职责要求：

1.    熟悉 Linux、windows、网络设备和运维

2.    熟悉应用服务、系统的深度安全加固

3.    熟悉恶意代码检测，清除

4.    熟悉常见漏洞攻击原理和入侵方式

5.    熟悉 python、shell 脚本语言，有脚本编写经验

6.    有工作热情，耐心、责任心强，沟通能力强

优先条件：

1.     有安全运维经验或应急响应经验优先

2.     善于探索，总结，优化流程优先

待遇：15 – 20k

安全实习生

岗位职责：

1.    参与公司安全系统 的日常运维、规则更新工作

2.    参与公司的安全系统的开发工作

3.    参与公司的安全运维工作，包括参与应急响应工作

职位要求：

1.     了解安全原理，与常见攻击方法

2.     对安全有热情，有责任心

3.     熟悉 linux、windows 系统

4.     实习时间可以保持每周3-4天，最少3个月

优先条件：

1.     熟悉正则表达式编写，熟悉linux系统，有脚本编写经验

2.     熟悉 python、java ，有开发经验

3.     熟悉 web 安全攻防，有渗透测试、安全测试经验

待遇：5K

福利待遇

1.    薪酬：15 – 35K（详见具体岗位薪酬），12月加2-6个月年底奖金

2.    每年定期调薪，优异者特殊调薪机制

3.    员工持股激励机制

4.    内部食堂，免费现磨咖啡、可乐等饮料

5.    每年国内、国外旅游，不定期的部门团建

6.    生日趴、相亲趴、各种游戏内测、福利大礼包

7.    无限制购买学习、研究资料、设备

8.    全球化公司，有外派出国（旧金山、阿姆斯特丹、马来西亚）等机会

工作地点：北京 

联系方式：heyi@pwrd.com（注明来自FreeBuf）
