卡巴斯基怒呛微软！指责Windows10实施杀毒软件不公平竞争

2016年11月15日发布

首页 ／ 行业 ／ 正文
3,213
0
0

导语：俄罗斯著名杀毒软件厂商卡巴斯基公开指控微软利用操作系统捆绑杀毒软件，造成了安全软件市场的不公平竞争，卡巴斯基已经向欧盟等相关机构进行了申诉，要求他们迫使微软采取整改措施。

日前，俄罗斯著名杀毒软件厂商卡巴斯基公开指控微软利用操作系统捆绑杀毒软件，造成了安全软件市场的不公平竞争，卡巴斯基已经向欧盟等相关机构进行了申诉，要求他们迫使微软采取整改措施。

微软为了摆脱外界对于 Windows 不安全的质疑，在几年前就开始发展自己的防毒软件，从 Windows 7 时期，推出了Microsoft Security Essentials，也开始内建防火墙。不过当时Microsoft Security Essentials 的防毒功能还刚起步，因此也不被防毒软件厂商看在眼里。

不过，微软持续不断的在改进他们的防毒功能，乃至到 Windows 10 现在大家都知道的 Windows Defender，加上原本就有的内建防火墙，还有上网保护的 SmartScreen，这些都是一般消费者在 Windows 里头可以看到与安全相关的功能，甚至在一般安全评测也渐渐取得不错的成绩。而这时，防毒厂商看在眼里可就不是那么满意了。

卡巴斯基指责微软的强迫用户安装Windows Defender

卡巴斯基实验室创始人、俄罗斯亿万富翁Eugene Kaspersky（卡巴斯基）最近于11月10日在他的博客上发布了一篇文章，标题就是“到此为止，我受够了！”他表示最近很多使用者在抱怨，当他们的 Windows 10 升级到新的更新时，系统会自动停用不支持新版系统的防毒软件，然后强迫使用者自动启用 Windows Defender。

卡巴斯基表示，就算是使用者已经安装了兼容于新版本的防毒软件，微软预设的确是会把 Windows Defender 给关闭，但是界面上依然还是会有一个很醒目特大号的按钮，要你把 Windows Defender“Turn On”。

他表示，微软在系统中捆绑了杀毒工具Defender，这一举动违背了市场公平竞争，另外微软还给第三方的杀毒软件开发商制造了障碍，损害了他们的利益。

这位创始人表示，已经向欧盟和俄罗斯的反垄断机构提出了申诉，并且提出了多个约束微软的要求。

据报道，微软公司在操作系统中植入了一些打击恶意软件的功能，以便在用户并未安装第三方杀毒软件的情况下，能够对电脑进行一定程度的保护。

比如在Windows 8操作系统中，如果侦测到外部第三方杀毒软件，微软内置的产品将会停止工作，让位给第三方产品。微软这样做是考虑到了个人电脑厂商的利益。这些企业往往和外部的安全软件进行了预装合作，并且获得一定的预装收入。

但是，卡巴斯基的不满还没结束，他表示一般防毒软件都会在使用者的使用期限到期前，会跳出一个提醒告知使用者防毒软件即将到期，提醒使用者续约的视窗。但是微软把这个视窗也消灭了，而把信息隐藏在 Windows Security Center 的通知栏里，但是这个通知栏一般人根本很少会去读。

而一旦用户没有查看到通知信息，并未采取任何动作的话，微软将会让第三方杀毒软件彻底停止工作，让自家的Defender重新发挥作用。

卡巴斯基在博文中还“揭露”了微软的一些不公平竞争行为。比如如果Windows 10操作系统发现用户电脑中安装了不兼容的杀毒软件，则会在系统安装过程中自动卸载对方，即使用户在升级过程中选择保留原先的软件和个人数据，用户杀毒软件也将受到影响。

微软已经断了第三方安全软件的后路

卡巴斯基还指出，在微软发布Windows 10和后续的更新之后，第三方安全软件开发商已经没有机会开发出兼容系统的产品。据称，微软首先把新系统提供给特定用户参加的内部测试计划，而在短时间后很快把系统发行给最终消费者。由于这两个动作之间的时间太过短暂，导致外部杀毒软件厂商无法对产品进行兼容性修改。

他更表示，根据 Windows 的规范，一台电脑中只能装有一套防毒软件。但是，如果你打开了 Windows Defender，则可以安装两套。他引用了乔治欧威尔在“动物农庄”中的名言来讽刺这种现象：所有动物都是平等的，但某些动物比其它动物更平等。

卡巴斯基表示，消费者会想，如果 Windows Defender 真的那么好，那么或许我们可以不用安装其它防毒软件，直接使用 Windows Defender 就好？卡巴斯基表示，结果恰恰相反。他引用了 2016 年 8 月份的一份独立机构的报告指出，Windows Defender 的安全性要低于平均值。

卡巴斯基认为，不仅仅只是防毒软件，微软正利用它们在操作系统上的优势，打压所有独立的第三方程序开发者。而这最终将会使得使用者面临危险，举例来说，当防毒软件单一化，将让黑客更容易侵入使用者的电脑，因为他们只要对付一个软件就好。

卡巴斯基还表示，欧盟等反垄断部门应该要求微软，在发行给最终消费者之前，首先给外部杀毒软件厂商提供独立版本，让他们有足够的时间升级安全软件。他也认为，微软应该对目前的升级和卸载方式进行调整。
