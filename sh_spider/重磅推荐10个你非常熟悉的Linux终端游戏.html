重磅推荐：10个你非常熟悉的Linux终端游戏

导语：提到“Linux”，很多人脑海里涌现的是这样一个画面：坐在系统前的人傻傻地盯着屏幕前一行行代码……单调、平凡而又无聊，仿佛只有无尽的命令、命令、更多的命令。

提到“Linux”，很多人脑海里涌现的是这样一个画面：坐在系统前的人傻傻地盯着屏幕前一行行代码……单调、平凡而又无聊，仿佛只有无尽的命令、命令、更多的命令。

 

鉴于您的此种想法，小编特意做了此次专题，记住千万不要低估Linux终端的力量，因为它几乎包罗万象。说到娱乐，更具体地说是游戏，windows平台确实是一个比linux更好的平台，但这不代表linux下没有游戏。

如今已经有成越来越多的游戏开始面向Linux终端，众多的Windows游戏也已移植到Linux平台。接下来的文章中，我将为大家介绍面向Linux终端的十款最佳游戏。愿君喜欢!

1. 2048

还记得2048吗？最早于2014年3月20日发行的2048曾让多少人痴迷不已。好消息是，现在它有了Linux终端版本——the 2048.c。你需要做的就是移动数字使其达到2048。

而为了安装游戏，你需要在终端依次输入如下命令：

wget https://raw.githubusercontent.com/mevdschee/2048.c/master/2048.c

启动游戏，需要在终端输入：

gcc -o 2048 2048.c ./2048

2. Bastet（俄罗斯方块）

 

还记得那些年我们一起拼命玩过的俄罗斯方块吗？Bastet 就是 Linux 版的俄罗斯方块。其基本规则是移动、旋转和摆放游戏自动输出的各种方块，使之排列成完整的一行或多行并且消除得分。运行时用空格键来翻转方块和用箭头来改变位置。

安装 Bastet，在终端输入：

sudo apt-get install bastet

启动游戏，则在终端输入：

bastet

注：如果启动时出现如下显示：

命令 'ninvaders' 可在 '/usr/games/ninvaders' 处找到；
由于/usr/games 不在PATH 环境变量中，故无法找到该命令；
ninvaders：未找到命令；

则可以直接输入/usr/games/ninvaders命令启动

3. Greed

 

在这个游戏中，你的当前位置由一个闪动的“@”表示，周围都是数字，你可以随意往上下左右四个方向移动。你选择的方向上，会有一个数字，这个数字的大小就会是你移动的步数。接着重复选择方向，但是你不可以移动到你已经去过的地方。到最后，当你不可以移动的时候，游戏结束！

安装 GREED，在终端中输入：

sudo apt-get install greed

启动 GREED，在终端中输入：

greed

4. Moonbuggy（赛车月球探险）

Moonbuggy是一款简单而又不失有趣的终端游戏，在这个字符图形的世界里，你驾驶着小巧的臭虫汽车飞奔在月球表面，当遇到星罗棋布的月亮环形山时，需要使用“空格键”跳过陨石坑，使用“a/i”开火。听起来很简单不是吗？等你一起来试试！

安装游戏，在终端输入：

sudo apt-get install moon-buggy

启动游戏，在终端输入：

moon-buggy

注意：Moonbuggy可以在相当宽的屏幕中运行，不过当屏幕宽度设置在小于80个字符的时候，可以取得较好的效果。

5. nInvaders（空间侵略者）

 

不知道你是否听说过“Space Invaders”，其实nInvaders就是基于Linux终端的“Space Invaders”。游戏中你需要使用“箭头键”和“空格键”从一波又一波的外星攻击中拯救地球，其中方向键用于移动飞船，空格用于射击外星人，可谓是最好玩的街机游戏之一。

安装 NINVADERS，在终端输入：

sudo apt-get install ninvaders

启动游戏，在终端输入：

ninvaders

6. Nsnake（贪吃蛇）

你知道20世纪末的那些古老的诺基亚手机上最棒的东西是什么吗？ 贪食蛇! 我以前在这个看似无聊但却让人上瘾的游戏上花费了大把的时间。在古老的诺基亚手机被智能手机取代的同时，贪食蛇也被另外的无聊但却令人上瘾的游戏取代了，比如说：神庙逃亡。但是，贪食蛇本身的魅力仍在。

在Android，iOS以及Web上有很多可以玩的贪食蛇游戏。另外，在Linux终端上也有类似的游戏。对，你没有听错，Linux终端上真的有。

nSnake 是Alexandre Dantas开发的经典游戏——贪食蛇的Linux终端版本。差不多在所有的Linux发行版中，你都可以获取nSnake。运行时，使用方向键来移动小蛇。

安装 NSNAKE，在终端输入：

sudo apt-get install nsnake

启动游戏，在终端输入：

nsnake

7. Pacman4Console（街机之王）

 

Pacman4console 是 Linux 版的街机之王，也是我最喜欢的游戏之一。在Pacman4console游戏中，你只需要用方向键来进行导航并吃掉所有的球，不让其被攻击者抓住。

安装 pacman4console，在终端输入：

sudo apt-get install pacman4console

启动游戏前，建议你先把终端窗口放大，然后输入：

pacman4console

8. Sudoku（数独）

 

还记得那段在各个报纸上搜索数独题来做的时光吗？现在有了基于Linux终端的Sudoku游戏，你就可以随时随地开始挑战了。此款游戏需要你用1—9中的数字填满整个9×9的网格，使其每一行、每一列以及每个3×3的正方形框内都包含从1—9的全部数字。

此外，这款基于Linux终端的游戏还可以为您提供不同难度水平，你可以选择简单或是困难模式进行挑战。

安装 Sudoku，在终端输入：

sudo apt-get install sudoku

启动游戏，在终端输入：

sudoku

9. Nethack

有点类似于“龙与地下城”，Nethack是极受欢迎的一款冒险游戏。名非其实，NetHack 并不是一款网络游戏。它只不过是基于一款出现较早的名为 Hack 的地牢探险类游戏开发出来的，而这款 Hack 游戏是 1980 年的游戏 Rogue 的后代。游戏需要寻找护身符 Yendor，而你的目标就是找到它并将它带给神。

安装 Nethack，在终端输入：

sudo apt-get install nethack-console

启动游戏，在终端输入：

nethack

10. Robot Finds Kitten（机器人发现小猫）

这款基于Linux终端的游戏将赋予你一个机器人的角色，而你的任务就是从无数的随机字符中尽可能多的找到小猫。你可以使用箭头键来进行浏览，别忘了看看对你遇到的每个对象的有趣描述。

安装 Robot Finds Kitten，在终端输入：

sudo apt-get install robotfindskitten

启动游戏，在终端输入：

robotfindskitten
