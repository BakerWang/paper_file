为什么Windows 10有着无与伦比的安全防护体验？

2016年12月22日发布

首页 ／ 新闻 ／ 正文
3,496
0
0

导语：现在人们一提到系统的安全性，首先会联想到Mac系统和Linux系统，而绝非Windows系统，为什么人们会对Windows的安全性有如此差的体验呢，这还要历史的看问题。

作为一个使用Windows系统的老司机，从Windows XP、Vista、Windows 7、Windows 8直至现在的Windows 10，能有的我都统统的使用了一遍，当然各个系统的使用感觉是完全不一样的，要论系统的体验，没有哪一个能比Windows XP经典，不过要论系统的安全性，就属Windows XP最差了。

微软系统更新史

现在人们一提到系统的安全性，首先会联想到Mac系统和Linux系统，而绝非Windows系统，为什么人们会对Windows的安全性有如此差的体验呢，这还要历史的看问题。早在2001年，微软就推出了后来被广受的争议的Windows XP，当时的微软对安全还未足够重视，更多关注点在满足用户痛点等体验层面，这导致Windows XP的安全性堪忧。在2003年和2004年，著名的“冲击波”和“震荡波”病毒肆虐，感染了上千万台机器，给微软用户带来了很大损失，也影响了微软的品牌信任度。另外微软没有很好的为其系统防护也是由于像McAfee和Symantec这样的安全厂商控告微软内置的。

最新的Microsoft安全智能报告显示在引入Windows Defender后，防病毒软件的使用有所增加，并强调Windows 10中的额外安全性改进防病毒功能违反了反垄断法。

不管怎么样，随着微软本身对安全防护的重视，目前Windows 10的安全防护体验号称有史以来最安全的 Windows，其内置的安全保护，将让你免受病毒、网络钓鱼和恶意软件的困扰。全新的安全防护功能已经通过自动更新进行升级，将帮助你保持你的操作系统始终处于最安全的状态，让你可以更加自由、放心的在 Windows 10 上做你所想做的。

根据最新的微软安全情报报告（第21卷）,如今还有大约28％的Vista PC和超过20％的Windows 7 PC仍然没有受到安全保护（这其中还包括许多没有统计进来的用户比如安装了盗版系统的用户），因为他们还没有把系统升级到Windows 10。

为什么在人们都高度重视网络安全的时代，这个数字还这么高？一个是许多用户已经长期的使用了像AVG，Avast这样的免费病毒防护软件，他们认为有这些软件在，他们的系统安全就会有保障，还有另外一个原因是由于世界上有些地方的网络不太发达，人们无法联网进行系统更新，比如在利比亚，阿尔及利亚，尼日利亚，伊拉克和坦桑尼亚这样的地区。

目前使用Windows 7的用户，有超过60％的PC仍然没有安装任何防病毒软件，还有20％的情况下虽然已经安装了但却把这些软件给关闭了。

微软的安全探索之路 

尽管人们对微软安全认知一直被掩盖在其产品功能的光环之下,但是贯穿于软件产品生命的信息安全却一直是微软公司战略的重中之重。微软自Windows 7开始就内置Windows Defender防御工具，但是Windows 7并不会系统一装上就自动开启Windows Defender，导致很多用户都没意识的到Windows Defender的存在。从Windows 8开始，微软才赋予了Windows Defender随机启动，正式担负起系统的防御工作，但功能相对较弱，和第三方标准框架的杀毒软件还是有一定的差距。如今微软操作系统Windows10已经发布一年多了，是什么让Windows 10有如此自信的安全体验呢？

1.Windows Hello

你有多少个密码？你几乎都不需要思考就能脱口而出“太多了！”那么 Windows Hello 将能帮你解决这一难题，Windows Hello是一种生物特征授权方式，用户只需要露一下脸，动动手指，就能立刻被运行Windows10的新设备（需要特殊硬件配置）所识别，实时访问自己的Windows 10设备。此外，Windows Hello可以提供企业级的安全保障，让用户真正摆脱密码，超越安全。

在你登录 Windows 之后做的第一件事情是什么？绝大多数人的选择是先打开浏览器。为了给用户提供更好的网络体验，Windows 10 上内置了全新的 Microsoft Edge 浏览器，这是一个更易于分享、阅读和研究浏览器，帮助你把事做好。从一开始，Microsoft Edge 就被设计成有史以来最安全的浏览器，它使用更高级的沙盒技术 （sandboxing technology），在线上浏览过程中，将你的个人信息、数据以及 Windows 隔离开。

2.SmartScreen

微软 SmartScreen 将帮助你在使用 Microsoft Edge 的时候，避开那些试图窃取你身份和个人信息的钓鱼网站。任意一个你访问的网站，微软 SmartScreen 都将检测它的信誉，并且屏蔽掉其中的恶意页面。在一个云端应用的帮助下，通过对每年发布在互联网上数十亿应用进行连续不断的信誉评估，SmartScreen 能够帮助你及时拦截那些社交网站产生的恶意软件。

3.Windows Defender

微软 SmartScreen 在你处于在线浏览的时候提供对外的安全保护，而 Windows Defender 则是在面对高级恶意软件的时候，帮你进行强有力的拦截。Windows Defender 是 Windows 10 上健全的反恶意软件解决方案中的一部分，它将让你的设备始终处于受保护状态。Windows Defender 将提供你所需要的安全防护，并处于不断更新中，以应对不断变化的病毒威胁，它将在第一时间快速检测、识别，以保护你的设备免受恶意软件的威胁。利用云计算的高度可拓展性和空间分布式后端服务，Windows Defender 能够在数百毫秒内，利用云端数十亿设备的数据，对恶意软件进行快速的分析并且报告给用户。

不过任何事情都需要一个发展过程，就目前Windows Defender的防护能力而言，还暂时和其他专业的杀毒软件有一定的差距,在今年4月份的AV-Test的杀软PK中在保护能力上，Avast、Bitdefender和F-Secure排名前三，Macfee、Windows Defender、Quick Heal排名垫底。但不可否认的一点就是作为防御工具Windows10上的Defender已经有了非常多的提升，而且也会随着Windows10系统的不断更新而得到功能加强。所以小编还是建议大家在使用Windows 10时先安全一些其他的防病软件，不过在这点上，微软又犯了反垄断的老毛病了，这不Windows 10内置的Windows Defender则让第三方反病毒软件厂商感到头疼，因为微软正在鼓励用户使用这款内置的安全程序而不要使用其他反病毒软件。在Windows 10上如果用户安装了第三方的反病毒软件，则Windows Defender始终会显示黄色的警告色。

微软Windows Defender会强制用户卸载其他AV软件

正如图中微软写的文字标识那样，微软正在鼓励用户卸载掉其他AV程序以便启用Windows Defender的实时保护。当用户点击黄色的打开按钮时会转到所有已安装程序列表，即用户可以在这里卸载掉第三方的AV程序。

如果只是这样的话那么用户不手动打开Windows Defender也倒是看不到这个提示，对于第三方AV程序来说似乎也没什么影响。

但是Windows 10的Windows Defender会定期（3~4天左右）在通知中心提示用户需要进行扫描，当用户点击通知就会打开这个界面。

所以诸如卡巴斯基等安全厂商就已经开始质疑微软的这一做法了,卡巴斯基实验室创始人、俄罗斯亿万富翁Eugene Kaspersky（卡巴斯基）日前发表博文，抨击微软在Windows 10上强制捆绑杀毒软件的做法。

他本人也表示，自己已向欧盟和俄罗斯的反垄断机构提出了申诉，要求约束微软的行为。因为免费升级，Windows 10已经覆盖4亿用户，并将在未来很长一段时间成为家庭用户桌面的主力，但卡巴斯基发现，目前三方杀软已经很难与新系统保持兼容，Windows 10频繁的更新和内测预览版，打乱了它们测试的节奏，还诱导了用户的负面吐槽。

比如在Windows 10 Build 14959中，就明确不兼容三方杀软。
