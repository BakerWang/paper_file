基础软件工程中心

中级JAVA开发工程师（2人）

岗位职责：

1. 按产品或项目要求进行技术开发，能独立完成模块分析设计；

2. 根据开发规范与流程独立完成详细设计，编码，测试及相关文档；

3. 负责系统设计、开发工作及项目开发框架的搭建和维护。

任职要求：

1. 本科及以上学历，计算机相关专业，3年及以上相关经验；

2. 3年及以上Java开发经验；

3. 精通B/S架构、JAVA编程语言、软件开发流程，熟悉软件工程；

4. 熟悉Html、JavaScript、JQuery、Ajax、JSON等前端技术；

5. 熟悉Oracle、MySQL、SQLServer，熟悉基本的调优方法；

6. 熟悉Redis，MongoDB等一种NoSQL数据库；

7. 有过大型互联网应用的开发经验，熟悉缓存架构，分布式架构，了解集群、负载均衡和容灾。

高级JAVA开发工程师（1人）

岗位职责：

1. 负责系统架构设计，开发架构，业务架构的规划。

2. 负责系统设计和核心模块代码编写，审查开发代码等工作。

3. 探索升级和改进产品的技术架构所需的技术，并给出具体的解决方案或应用框架。

任职要求：

1. 本科及以上学历，计算机相关专业，5年及以上相关经验；

2. 熟悉各主流架构，熟悉Linux系统，能针对应用系统需求，进行基本系统网络调优；

3. 熟悉Html、JavaScript、JQuery、Ajax、JSON等前端技术；

4. 熟悉Oracle、MySQL、SQLServer，熟悉基本的调优方法；

5. 熟悉Redis，MongoDB等一种NoSQL数据库；

6.有大数据处理经验优先；

7.有过大型互联网应用的开发经验，熟悉缓存架构，分布式架构，了解集群、负载均衡和容灾。

UI设计师（2人）

岗位职责：

1.负责PC端、移动终端等界面的UI设计；

2.负责制定设计规范、风格模版及设计趋势研究；

3.负责创新性项目的研究及概念设计；

4.通过视觉表现吸引用户使用产品。

任职要求：

1.设计、美术相关专业,3年或以上相关经验；

2.精通常用的图像处理工具(Photoshop/illustrator等）,熟悉前端HTML、CSS更佳；

3.熟悉界面设计的流程方法，出色的设计语言表达能力，优秀的创新与沟通协调能力；

4.良好的沟通与设计表达能力，具有责任心，良好的团队协作，执行力与学习能力。

前端开发工程师（1人）

岗位职责：

1. 负责桌面端、移动端H5页面、微信小程序等页面开发优化工作；

2. 优化网站前端功能设计，解决各种浏览器的兼容性问题

3. JavaScript程序模块开发，通用类库、框架编写；

4. 通过各种前端技术手段，提升交互效果和用户体验并满足性能要求。

技能要求：

1. 具备扎实的HTML、JavaScript、CSS基础，熟悉主流的前端框架，如jQuery / Vue/ React / Angular等；

2. 熟悉前后端交互流程，并熟知移动端的开发技巧；

3. 熟悉前端构建工具，如webpack、FIS，若对前端工程化有自己的实践者优先；

4. 熟悉浏览器内部工作模式，了解主流浏览器引擎（WebKit, Gecko, Trident等）熟悉设计模式，熟练掌握面向对象编程和事件驱动编程风格。

产品经理(2人)

岗位职责：

1. 进行市场调研、用户需求分析、用户模型建立，研究产品发展趋势，进而对整个产品进行市场定位与整体规划；

2. 负责产品规划、业务流程设计、功能设计、产品优化等工作；

3. 制定产品需求计划,进行可行性分析和概要设计,撰写用户需求说明书,负责需求的跟踪和控制；

4. 了解用户需求，分析竞争对手动态和市场动态，执行产品线路图，提出产品需求和改良意见，形成原型图和需求策划文案等工作；

5. 协调各方资源，指导和督促产品开发过程，保障产品最终的进度与质量。

任职要求：

1. 3年以上互联网产品经理经验，有电商、分享经济等产品经验优先；

2. 熟悉产品实施过程，包括市场分析、需求分析、产品功能设计、业务流程设计、界面设计、用户研究和可用性测试等；

3. 有至少1款以上App或者网站的产品管理经验，能够及时跟踪用户反馈，总结并实现迭代优化；

4. 具备用户同理心，敏锐的市场嗅觉，良好的运营意识与创新意识；

5. 需求分析、需求文档编写、AXURE、visio、xmind设计等基本技能牢固；

6. 对数据敏感，具备较强分析加工和沟通能力；

7. 积极上进，能承受较大的工作压力。

智慧城市工程中心

咨询规划工程师（1人）

岗位职责：

1. 根据项目要求进行项目调研，挖掘客户需求；

2. 结合实际情况进行项目咨询方案设计，并作出项目预算；

3. 能独立完成项目的调研、协调工作，以及方案的编制工作；

4. 领导安排的其他工作。

任职要求：

1. 本科及以上计算机、通讯相关专业、5年以上工作经历；

2. 具有良好的沟通能力、团队合作能力；

3. 具有技术方案编写及整合能力；

4. 有智慧城市、电子政务等相关领域从业经验优先。

售前助理实习生（1人）

岗位职责：

1. 负责项目相关数据的收集、整理和分析工作；

2. 参与项目调研及需求分析，编写需求文档等；

3. 参与售前、咨询工作，保证项目正常实施；

4. 负责为客户提供现场实施、培训及技术支持工作；

5. 完成上级主管交办的其他任务。

任职要求：

1. 本科及以上学历，有扎实的文字功底、有较强的逻辑思维和快速学习能力；

2. 热爱IT咨询规划行业，善于沟通，有强烈的团队合作精神；

3. 保证能实习半年以上；

4. 适应较大工作压力，善于自我调节。

电子数据取证实验室

电子数据取证研究人员（2-3人）

岗位职责：

1. 承担电子数据取证和司法鉴定相关科研项目的研究工作；
2. 参加电子数据取证与司法鉴定的实际工作；
3. 参加电子数据取证与司法鉴定方法与标准和技术规范的研究；
4. 参与申请电子数据取证与司法鉴定相关的科研和技术开发项目。

岗位要求：

1.了解电子数据取证领域研究的热点难点问题,掌握基本理论和基本知识,熟悉研究方法和实验方法；
2.掌握相关的电子数据司法鉴定/检验程序、方法、规程及结果判断标准或依据，熟悉电子数据鉴定领域仪器设备的性能、状况、操作维护方法和安全规程；
3.性格成稳、诚实守信、工作踏实勤奋，有责任心，具有较强的协调能力与团队协作精神；
4. 电子信息技术或司法鉴定领域硕士以上学历，工作经验不限，有相关领域科研成果并有中、高级职称者优先考虑。

电子数据司法鉴定助理（2人）

岗位职责：

1. 协助司法鉴定人完成司法鉴定工作；

2. 从事电子取证相关的项目研究工作；

3. 完成司法鉴定所和实验室交办的工作。

职位要求：

1. 电子取证相关专业毕业或具有相关专业的知识基础；
2. 了解电子取证的法律法规和技术规范及基本流程；
3. 遵守实验室和司法鉴定所的管理规定；
4. 工作认真细致，严格按照技术流程和管理规范从事司法鉴定工作和科学研究工作；
5. 具有团队合作精神，易于沟通交流；
6. 能吃苦耐劳，能适应高强度的工作；
7. 熟练的中英文口语及写作能力；
8. 有硕士及以上学位者优先。

电子数据司法鉴定实习生（3人）

岗位职责：

1. 参加导师的研究项目,完成老师交办的辅助性工作；
2. 配合项目负责人完成辅助性工作；
3. 从事一些项目文档的编写、整理等工作；
4. 整理和完善研究项目或对外服务的有关材料。

职位要求：

1. 本科大四在读学生或在读硕士、博士研究生均可；

2. 电子数据取证或司法鉴定相关专业；

3. 保证至少6个月的上班时间；

4. 有良好的团队协作精神，较强的理解能力和沟通能力；

5. 网络信息安全与电子取证方向及政法大学电子数据司法鉴定专业或刑事科学技术专业优先考虑。
