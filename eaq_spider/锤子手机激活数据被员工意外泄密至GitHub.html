锤子手机激活数据被员工意外泄密至GitHub

  • 快科技
    • /
  • 数据泄露
    • /
  • 2017-07-13 12:53

罗永浩和他的锤子/坚果手机绝对是手机圈的另类，独特的情怀笼络了大量的忠实拥趸，但商场是无情的，一切都要靠销量来说话，而最新的坚果Pro不小心遭遇了一个沉重打击。

坚果Pro发布后，罗永浩曾经坚称备货40万台(相关微博后被删除)，而在正式发售后，有说法称这首批40万台迅速售罄，而且几乎每天都能卖2万台以上，但都无从证实。

结果没想到，秘密被锤子自己的员工给捅破了。

坚果

近日，锤子员工李貌在GitHub上提交的文件被发现竟然包含坚果Pro的激活数量，截止6月15日共计207707行数据，也就是说，坚果Pro迄今的销量还不到21万台。

基本可以证实，当事人确实是锤子员工，而且这个数字和京东上坚果Pro不同型号的评论分布基本一致，说明是靠谱的。

泄密文档

关键是，泄露文件中详细罗列了每一台已激活坚果Pro手机的SN号、IMEI号、版本，如此程度的泄密简直不可思议、闻所未闻。

有网友验证后发现，自己的手机确实在其中…… 

泄密文档

虽然相关文件很快被删除，但证据已经被网友保存了下来，可以看到坚果Pro最畅销的是碳黑色64GB，卖了8.4万部，占比为40.49％，其次是细红线128GB，接近5.1万部，占比接近25％，然后是酒红色64GB，3.3万部占比16％。

分析发现，坚果Pro卖得最好的是发布会之后的几天，每天都能卖一两万台，然后是每隔6天现货一次，也就会出现一个销量高峰，但直到6月1号前，销量一直在降低，不过618期间还是不错的，每天都能卖上几千台。

618后又不行了，7月份现后略有回升，但从7月9号往后销量又开始下降了，而现在所有SKU都现货的情况下，销量还在下滑，说明是真的卖不出去了，那么要想增加销量，只有一个办法，那就是降价。

所以7月10日起购买64GB版送99元礼品，购买128GB版送200元礼品，已经变相降价了。

补充报道：

锤子相关人员对这条消息做出了回应，称该数据只是推动调研的活跃用户数量，并非是目前整个的销量。所谓的“推动调研的活跃用户数量”，该人员称是锤子进行NPS用户调研，初筛出来对系统升级比较积极的群体，不是销量。

不管约21万这个销量数字靠不靠谱，坚果Pro的销量都已经远超T1、T2和M1系列同时期的销量了。
